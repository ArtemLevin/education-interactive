Средняя скорость

Весь путь разделить на всё время. Не путать со средним арифметическим скоростей!

📐

Обратные задачи

Нахождение неизвестного числа по известному среднему и остальным числам.

🔄

Группы чисел

Как найти общее среднее, если известны средние нескольких групп чисел.

1

Что дано

Пешеход за первый час прошёл 5,7 км, за второй час — 5,4 км, за третий час — 4,8 км.
v₁ = 5,7 км/ч   v₂ = 5,4 км/ч   v₃ = 4,8 км/ч
Каждый отрезок времени равен 1 часу. Расстояния за каждый час численно равны скоростям.
2

Что нужно найти

Найти среднюю скорость пешехода за всё время движения.
Важно: это не среднее арифметическое скоростей, а отношение всего пути ко всему времени!
Средняя скорость — это такая постоянная скорость, с которой нужно было бы двигаться, чтобы пройти тот же путь за то же время.
3

Ключевая формула

vср = Sобщ / tобщ = (S₁ + S₂ + S₃) / (t₁ + t₂ + t₃)
Формула работает для любого количества отрезков пути. Суммируем все расстояния и делим на сумму всех времён.
4

Вычисляем общий путь

Складываем все пройденные расстояния:
Sобщ = 5,7 + 5,4 + 4,8 = 15,9 км
5,7 + 5,4 = 11,1;   11,1 + 4,8 = 15,9
5

Вычисляем общее время

Складываем все затраченные времена:
tобщ = 1 + 1 + 1 = 3 ч
Каждый отрезок движения длился 1 час, всего 3 отрезка.
6

Находим среднюю скорость

Делим путь на время:
vср = 15,9 / 3 = 5,3 км/ч
Проверка: 5,3 × 3 = 15,9 — совпадает с общим путём. Ответ верен.

Ответ: 5,3 км/ч

✓

Итог и проверка

Средняя скорость 5,3 км/ч — это значение, которое находится между минимальной (4,8) и максимальной (5,7) скоростями.

Если бы пешеход шёл всё время со скоростью 5,3 км/ч, за 3 часа он прошёл бы 5,3 × 3 = 15,9 км — ровно столько, сколько прошёл в условии.

Визуализация

S₁ = 5,7 км (1 ч)
S₂ = 5,4 км (1 ч)
S₃ = 4,8 км (1 ч)
vср = 5,3 км/ч

Типичные ошибки

✕

Среднее арифметическое вместо средней скорости

Нельзя просто сложить скорости и поделить на количество отрезков, если времена разные. Всегда используйте Sобщ / tобщ.

✕

Деление на количество участков вместо общего времени

В знаменателе должно быть суммарное время движения, а не число отрезков пути.

✕

Ошибки с десятичными дробями

При сложении чисел с запятой внимательно выравнивайте разряды. 5,7 + 5,4 ≠ 10,11, а 11,1.

✕

Потеря единиц измерения

Всегда записывайте единицы: км/ч для скорости, км для расстояния, ч для времени.

Краткая памятка

Основные формулы

  • Среднее арифметическое: x̄ = (x₁ + x₂ + ... + xₙ) / n
  • Сумма через среднее: S = x̄ · n
  • Средняя скорость: vср = Sобщ / tобщ
  • Расстояние: S = v · t

Алгоритм решения задач на среднюю скорость

  • 1. Найти расстояние на каждом отрезке: Sᵢ = vᵢ · tᵢ
  • 2. Сложить все расстояния: Sобщ = ΣSᵢ
  • 3. Сложить все времена: tобщ = Σtᵢ
  • 4. Разделить: vср = Sобщ / tобщ

Проверка ответа

  • Средняя скорость должна быть между минимальной и максимальной скоростями на отрезках
  • Проверьте: vср · tобщ = Sобщ
  • Единицы измерения: км/ч для скорости, км для расстояния
