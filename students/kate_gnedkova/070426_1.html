📊 График рациональной функции

Задача: Постройте график функции y = ( x2 − 2 x − 3 ) ( x2 − 3 x + 2 ) x2 − 4 x + 3 и определите, при каких значениях m прямая y=m имеет с графиком ровно одну общую точку.
Настройки отображения
Идея решения

Стратегия решения:

  • Разложить числитель и знаменатель на множители (корни квадратных трехчленов)
  • Найти ОДЗ и сократить дробь
  • Определить координаты "выколотых" точек (где функция не определена)
  • Построить график полученной параболы с выколотыми точками
  • Проанализировать пересечение прямой y=m с графиком
Шаг 1: Разложение на множители
1 Найдем корни квадратных трехчленов:
x2 − 2 x − 3 = 0 ⇒ x1 = 3 , x2 = − 1
x2 − 3 x + 2 = 0 ⇒ x1 = 1 , x2 = 2
x2 − 4 x + 3 = 0 ⇒ x1 = 1 , x2 = 3
Шаг 2: ОДЗ и сокращение
2 Область допустимых значений:

Знаменатель не должен быть равен нулю:

x2 − 4 x + 3 ≠ 0 ⇒ x ≠ 1 , x ≠ 3

Сокращаем дробь:

y = ( x − 3 ) ( x + 1 ) ( x − 1 ) ( x − 2 ) ( x − 1 ) ( x − 3 ) = ( x + 1 ) ( x − 2 )

Итоговая функция: y=x2−x−2, при x≠1 и x≠3.

Шаг 3: Выколотые точки
3 Найдем координаты точек, которые нужно выколоть:
  • При x=1: y=12−1−2=−2. Точка (1; -2).
  • При x=3: y=32−3−2=4. Точка (3; 4).
Шаг 4: Построение графика
4 Строим параболу y=x2−x−2:
  • Ветви вверх.
  • Вершина: xв=−−12=0.5.
  • yв=0.52−0.5−2=0.25−2.5=−2.25.
  • Выкалываем точки (1; -2) и (3; 4).
Шаг 5: Анализ пересечений с прямой y = m
5 Когда прямая y = m имеет ровно одну общую точку?
  • 1. Касание вершины параболы: m=−2.25. (1 точка)
  • 2. Прохождение через выколотую точку (1; -2): m=−2.
    Проверим остальные пересечения при m = -2: x2−x−2=−2⇒x2−x=0⇒x=0 или x=1.
    x=1 выколота, остается x=0. Итого 1 точка. ✓
  • 3. Прохождение через выколотую точку (3; 4): m=4.
    Проверим остальные пересечения при m = 4: x2−x−2=4⇒x2−x−6=0⇒x=3 или x=−2.
    x=3 выколота, остается x=-2. Итого 1 точка. ✓
Ответ
Прямая y = m имеет ровно одну общую точку с графиком при:
m ∈ { − 2.25 ; − 2 ; 4 }
Проверка ответа

Проверим другие значения m:

  • m=−3 (ниже вершины): 0 точек.
  • m=−2.1 (между вершиной и выколотой точкой -2): 2 точки.
  • m=0 (между выколотыми точками -2 и 4): 2 точки (корни параболы -1 и 2).
  • m=5 (выше выколотой точки 4): 2 точки.
Типичные ошибки

⚠️ Частые ошибки при решении:

Ошибка 1: Забыли найти ОДЗ и не выкололи точки (1; -2) и (3; 4).
Ошибка 2: Неправильно разложили квадратные трехчлены на множители.
Ошибка 3: Не проверили, сколько точек остается при прохождении прямой через выколотые точки (думают, что там 0 точек).
Ошибка 4: Перепутали знаки при нахождении вершины параболы.
