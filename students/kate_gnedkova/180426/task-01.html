Уравнение с вынесением общего множителя и формулой квадрата суммы

Задача учит не раскрывать всё подряд, а сначала искать структуру выражения и повторяющийся множитель.

Что дано

  • Нужно решить уравнение x(x2+6x+9)=4(x+3)

Что нужно найти

  • Найти все значения x, при которых равенство верно.

Смысл условия

  • Внутри первой скобки спрятан полный квадрат.
  • После этого уравнение удобно превратить в произведение.
  • Как только получилось произведение, включается правило нуля для множителей.

Идея решения

  • Узнать выражение x2+6x+9 как квадрат суммы.
  • Перенести всё в одну сторону и вынести общий множитель (x+3)
  • Решить два более простых уравнения.

Пошаговое решение

Шаг 1. Узнаём полный квадрат

Замечаем формулу

x2+6x+9=(x+3)2

Шаг 2. Переписываем уравнение

x(x+3)2=4(x+3)

Переносим правую часть влево:

x(x+3)2−4(x+3)=0

Шаг 3. Выносим общий множитель

(x+3)(x(x+3)−4)=0

Внутри второй скобки упрощаем:

(x+3)(x2+3x−4)=0

Шаг 4. Решаем по правилу произведения

Получаем две ветви:

x+3=0x2+3x−4=0

Из первой ветви:

x=−3

Вторая ветвь раскладывается на множители:

(x+4)(x−1)=0

Отсюда

x=−4,x=1

Визуальная опора

Проверка ответа

  • При x=−3 обе части равны нулю.
  • При x=−4 левая часть равна 4, правая часть тоже равна 4.
  • При x=1 левая часть равна 16, правая часть тоже равна 16.

Вывод

Ответ:

x∈{−4;−3;1}

Типичные ошибки

  • Раскрыть скобки сразу и получить длинное кубическое выражение.
  • Не заметить общий множитель (x+3)
  • После вынесения решить только одну ветвь и потерять часть корней.

Самопроверка

Проверьте себя коротким вопросом. Какой множитель выносится после переноса вправо влево?
