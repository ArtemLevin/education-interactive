📊 График функции с модулем

Задача: Постройте график функции y = | x | ⋅ ( x − 1 ) − 2 x и определите, при каких значениях m прямая y=m имеет с графиком ровно две общие точки.
Настройки отображения
Идея решения

Стратегия решения:

  • Раскрыть модуль, рассмотрев два случая: x≥0 и x<0
  • Построить график кусочной функции (две параболы)
  • Найти вершины парабол и точки пересечения с осями
  • Проанализировать, при каких m горизонтальная прямая пересекает график ровно в двух точках
Шаг 1: Раскрытие модуля
1 Что дано:
y = | x | ⋅ ( x − 1 ) − 2 x

Вспомним определение модуля:

Рассмотрим два случая для раскрытия модуля.

Функция содержит модуль |x|, поэтому её нужно разбить на две части в зависимости от знака x.

Функция содержит модуль |x|, поэтому её нужно разбить на две части в зависимости от знака x.

Почему это важно: Модуль меняет своё выражение в точке x=0, поэтому график функции будет состоять из двух различных частей, соединённых в этой точке.

Шаг 2: Случай x ≥ 0
2 При x≥0: |x|=x
y = x ⋅ ( x − 1 ) − 2 x = x2 − x − 2 x = x2 − 3 x

Это парабола, ветви вверх.

Найдём ключевые точки:

  • Корни: x2−3x=0⇒x(x−3)=0⇒x=0 или x=3
  • Вершина: xв=32=1.5
  • yв=(1.5)2−3⋅1.5=2.25−4.5=−2.25
Шаг 3: Случай x < 0
3 При x<0: |x|=−x
y = ( − x ) ⋅ ( x − 1 ) − 2 x = − x2 + x − 2 x = − x2 − x

Это парабола, ветви вниз.

Найдём ключевые точки:

  • Корни: −x2−x=0⇒−x(x+1)=0⇒x=0 или x=−1
  • Вершина: xв=−−12⋅(−1)=−12=−0.5
  • yв=−(−0.5)2−(−0.5)=−0.25+0.5=0.25
Шаг 4: Построение графика
4 Сводим всё вместе:
y = { −x2−x , при x<0 x2−3x , при x≥0

Ключевые точки графика:

  • (-1; 0) — корень левой параболы
  • (-0.5; 0.25) — вершина левой параболы (максимум)
  • (0; 0) — точка соединения
  • (1.5; -2.25) — вершина правой параболы (минимум)
  • (3; 0) — корень правой параболы
Шаг 5: Анализ пересечений с прямой y = m
5 Геометрическая интерпретация:

Прямая y=m — это горизонтальная прямая на высоте m.

Рассмотрим, сколько точек пересечения будет при разных m:

  • При m>0.25: прямая выше максимума левой параболы → 0 точек
  • При m=0.25: касается вершины левой параболы → 1 точка
  • При 0<m<0.25: пересекает левую параболу в 2 точках → 2 точки ✓
  • При m=0: проходит через корни (-1;0), (0;0), (3;0) → 3 точки
  • При −2.25<m<0: пересекает обе параболы → 2 точки ✓
  • При m=−2.25: касается вершины правой параболы → 1 точка
  • При m<−2.25: ниже минимума → 0 точек
Ответ
Прямая y = m имеет ровно две общие точки с графиком при:
m ∈ ( − 2.25 ; 0 ) ∪ ( 0 ; 0.25 )

Или в виде неравенств: −2.25<m<0 или 0<m<0.25

Проверка ответа

Проверим граничные значения:

  • m=0.2 (из первого интервала): прямая пересекает левую параболу в двух точках, правую не пересекает → 2 точки ✓
  • m=−1 (из второго интервала): прямая пересекает обе параболы по одной точке → 2 точки ✓
  • m=0 (граничное): проходит через три точки (-1;0), (0;0), (3;0) → 3 точки (не подходит) ✓
Типичные ошибки

⚠️ Частые ошибки при решении:

Ошибка 1: Забыли раскрыть модуль и построили одну параболу вместо двух
Ошибка 2: Неправильно определили знак перед x² во втором случае (должно быть -x²)
Ошибка 3: Не учли, что при m = 0 получается 3 точки пересечения, а не 2
Ошибка 4: Включили граничные точки m = 0.25 и m = -2.25 (в этих случаях только 1 точка)
Ошибка 5: Не нашли вершины парабол, из-за чего невозможно определить интервалы
