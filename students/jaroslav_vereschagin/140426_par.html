График параболы: влияние коэффициентов

Квадратичная функция y=ax2+bx+c — интерактивный конспект

МАТЕМАТИКА · НАГЛЯДНО · КВАДРАТИЧНАЯ ФУНКЦИЯ

Квадратичная функция — это функция вида:

y=ax2+bx+c

где a≠0, а a, b, c — числа (коэффициенты).

Графиком квадратичной функции является парабола.

💡 Если a=0, функция становится линейной (y=bx+c), и её график — прямая линия.

Коэффициент a определяет две вещи:

  • Направление ветвей: если a>0 — ветви вверх ↑, если a<0 — ветви вниз ↓
  • «Ширину» параболы: чем больше |a|, тем уже парабола; чем меньше |a|, тем шире
Вывод

a — главный коэффициент. Он отвечает за то, куда смотрит парабола и насколько она «узкая».

Коэффициент b влияет на положение вершины параболы по оси X:

x0=−b2a

Вершина параболы находится в точке (x0;y0), где y0=ax02+bx0+c.

💡 Знак b важен вместе со знаком a. Если a и b одного знака — вершина левее оси Y, если разных — правее.

Коэффициент c — это значение функции при x=0:

y(0)=a⋅02+b⋅0+c=c

Парабола всегда пересекает ось Y в точке (0;c).

Изменение c сдвигает параболу вверх-вниз, не меняя её формы.

Дискриминант показывает, сколько раз парабола пересекает ось X:

D=b2−4ac
  • D>0 — два корня, парабола пересекает ось X в двух точках
  • D=0 — один корень (вершина лежит на оси X)
  • D<0 — нет корней, парабола не пересекает ось X
1Определи направление ветвей по a

Если a>0 — ветви направлены вверх. Если a<0 — ветви направлены вниз.

Проверь знак первого коэффициента — это первое, на что нужно смотреть.
2Найди вершину параболы

Абсцисса вершины: x0=−b2a

Ордината вершины: y0=ax02+bx0+c

Вершина — самая важная точка параболы.
3Найди точку пересечения с осью Y

Подставь x=0: получаем y=c. Точка: (0;c).

4Вычисли дискриминант

D=b2−4ac

  • D>0 — два пересечения с осью X
  • D=0 — одно пересечение (касание)
  • D<0 — нет пересечений
5Построй параболу

Отметь вершину, точку (0;c), и если есть — точки пересечения с осью X. Проведи плавную кривую через эти точки.

Парабола симметрична относительно вертикальной прямой x=x0.
Главный вывод

Чтобы построить параболу, нужно знать: направление ветвей (a), вершину (x0=−b2a), точку на оси Y (c) и наличие корней (D).

Типичные ошибки

Ошибка 1: Путают направление ветвей

Ученик видит a=−3 и рисует ветви вверх, потому что «тройка большая».

✅ Как правильно: смотрите на знак a, а не на его модуль. Минус — ветви вниз.
Ошибка 2: Забывают минус в формуле вершины

Пишут x0=b2a вместо x0=−b2a.

✅ Как правильно: в формуле вершины всегда стоит минус: x0=−b2a.
Ошибка 3: Считают, что c — это вершина

Ученик думает, что c — это y-координата вершины.

✅ Как правильно: c — это точка пересечения с осью Y, то есть (0;c).
Ошибка 4: Не проверяют D перед нахождением корней

Сразу подставляют в формулу корней, не проверив дискриминант.

✅ Как правильно: всегда сначала считайте D=b2−4ac. Если D<0 — корней нет.
Ошибка 5: Забывают, что a ≠ 0

При решении задач с параметром забывают исключить случай a=0.

✅ Как правильно: если коэффициент при x² содержит параметр, рассмотрите случай, когда он равен нуль.

Памятка

Формулы

y=ax2+bx+c

Коэффициент a

  • a > 0 → ветви вверх ↑
  • a < 0 → ветви вниз ↓
  • |a| ↑ → парабола уже
  • |a| ↓ → парабола шире

Коэффициент b

  • Влияет на x₀ вершины
  • x0=−b2a
  • a, b один знак → x₀ < 0
  • a, b разный знак → x₀ > 0
  • b = 0 → вершина на оси Y

Коэффициент c

  • Точка (0; c) — пересечение с OY
  • Сдвиг вверх-вниз
  • Не влияет на форму
  • Не влияет на направление

Дискриминант

  • D > 0 → 2 корня
  • D = 0 → 1 корень (касание)
  • D < 0 → 0 корней

Алгоритм построения

  • 1. Определи направление (a)
  • 2. Найди вершину (x₀, y₀)
  • 3. Отметь (0; c)
  • 4. Найди корни (если D ≥ 0)
  • 5. Проведи параболу

Итог

Ключевые идеи

  • Квадратичная функция y=ax2+bx+c задаёт параболу
  • Коэффициент a — направление и ширина ветвей
  • Коэффициент b — горизонтальное смещение вершины
  • Коэффициент c — вертикальный сдвиг, точка пересечения с осью Y
  • Вершина параболы: x0=−b2a, y0=ax02+bx0+c
  • Дискриминант D=b2−4ac определяет число корней

Что нужно уметь

  • Найти координаты вершины параболы

Связь коэффициентов с графиком

Изменение a растягивает/сжимает и переворачивает параболу. Изменение b сдвигает вершину по горизонтали. Изменение c сдвигает всю параболу строго вверх или вниз.
