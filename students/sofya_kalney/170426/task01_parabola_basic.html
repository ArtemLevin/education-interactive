Задание 1. Чтение коэффициентов квадратной функции

Разбираем функцию y = x² − 4x + 1: как по формуле быстро понять знак коэффициентов и положение вершины.

← Ко всем заданиям

Что дано

Функция: y = x² − 4x + 1
Найти
знак a, знак b, значение c, положение вершины
Ключ
y = ax² + bx + c
Формула вершины
xв = −b/(2a)

Идея решения

У квадратной функции каждый коэффициент отвечает за конкретную геометрическую характеристику графика. Мы просто «считываем» формулу слева направо.

Шаг 1
Сравниваем y = x² − 4x + 1 со стандартным видом y = ax² + bx + c. Получаем: a = 1, b = -4, c = 1.
Шаг 2
Так как a = 1 > 0, ветви параболы направлены вверх.
Шаг 3
Коэффициент b = -4 отрицательный. Это уже отвечает на второй вопрос.
Шаг 4
Коэффициент c = 1 — это ордината точки пересечения с осью Oy. Значит, график проходит через точку (0;1).
Шаг 5
Находим абсциссу вершины: xв = -(-4)/(2·1) = 2. Число положительное, значит вершина расположена правее оси Oy.
Проверка
Все ответы согласованы: ветви вверх, вершина справа, точка пересечения с Oy имеет ординату 1.

Вывод

  • a > 0;
  • b < 0;
  • c = 1;
  • вершина правее оси Oy.

Типичные ошибки

Часто путают: c — это не координата вершины, а значение функции при x=0. Ещё одна ошибка — определять положение вершины «на глаз», не используя формулу xв = -b/(2a).

Самопроверка

Вопрос. У функции y=x²+6x-2 вершина левее или правее оси Oy?

Интерактивная модель

Ползунок меняет коэффициент b и показывает, как двигается вершина. Базовое задание подсвечено отдельно.
