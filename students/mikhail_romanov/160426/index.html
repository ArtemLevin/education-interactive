Сечения многогранников

Это многостраничный учебный сайт по пособию: каждая страница посвящена одному уникальному заданию из тренировочного блока. Внутри каждой страницы есть пошаговый разбор, обоснование, типичные ошибки, наглядная схема, самопроверка и удобная навигация.

Как пользоваться сайтом

1. Начните с задачи

Откройте любую страницу и сначала прочитайте блок «Что дано» и «Ключевая идея».

2. Смотрите на схему справа

Она помогает увидеть переходы между гранями и рёбрами, а не просто запомнить картинку.

3. Проговорите доказательство

После построения обязательно ответьте себе: почему новая точка принадлежит плоскости сечения?

4. Проверьте себя

На каждой странице есть короткая самопроверка и памятка по типичным ошибкам.

Краткий алгоритм темы

  1. Найти две точки, которые точно лежат в одной грани.
  2. Провести через них первый след секущей плоскости.
  3. Продлить его до общего ребра нужных граней.
  4. Получить новую вершину сечения как пересечение следа с ребром.
  5. Повторять переходы между гранями, пока контур не замкнётся.
  6. В финале проверить, что каждый отрезок действительно лежит в своей грани.
