Наглядная геометрия: плоскости, углы и сечения

Большая интерактивная шпаргалка по стереометрии: как составлять уравнение плоскости, как извлекать из него нормаль, как находить угол между плоскостями и как считать площадь сечения через его проекцию на основание.

Схема решения

В этой теме почти всё держится на одной связке: удачно выбрали систему координат → составили уравнение плоскости → сразу получили нормаль → угол между плоскостями свели к углу между векторами → при необходимости нашли площадь сечения через проекцию.

Пошаговое решение

Идея решения

Вместо того чтобы сразу охотиться за углом или площадью, мы сначала строим координатную модель. Это делает задачу более алгебраической и управляемой. Особенно полезен приём: поставить начало координат в точку, лежащую в нужной плоскости. Тогда свободный коэффициент часто исчезает, потому что получается D = 0.

Главная мысль. Уравнение плоскости — это не конечная цель, а инструмент. Из него достаётся нормаль, а нормаль уже даёт угол между плоскостями. Потом этот угол используется в формуле площади сечения через проекцию.

Ключевые формулы

Ax + By + Cz + D = 0
Общее уравнение плоскости
n = (A, B, C)
Нормаль к плоскости
cos φ = |n₁ · n₂| / (|n₁| · |n₂|)
Угол между плоскостями равен углу между их нормалями; берём острый угол
Sсеч = Sпр / cos φ
Площадь сечения через площадь его ортогональной проекции на основание

Типичные ошибки

  • Писать уравнение плоскости как Ax + y + z + D = 0 вместо Ax + By + Cz + D = 0.
  • Думать, что коэффициент D обязан быть только 0 или 1. Это неверно: он может быть любым числом.
  • Забывать модуль в формуле для косинуса и получать тупой угол вместо острого.
  • Искать площадь сечения напрямую, хотя проще сначала найти площадь проекции.
  • Путать саму фигуру сечения и её проекцию на основание.
Проверка здравого смысла. Если найденный косинус по модулю больше единицы, значит где-то ошибка в вычислениях. Если проекция по площади больше самого сечения при наклонном положении, тоже ошибка.

Самопроверка

Чек-лист после уравнения плоскости
  • Все нужные точки подставлены?
  • Уравнение удовлетворяется для каждой точки?
  • Коэффициенты сокращены красиво, но без потери смысла?
Чек-лист после угла
  • Нормали выписаны верно?
  • Скалярное произведение посчитано без ошибки знака?
  • В ответе взят именно острый угол?

Памятка для печати

  • Уравнение плоскости: Ax + By + Cz + D = 0.
  • Нормаль: (A, B, C).
  • Если плоскость проходит через начало координат, то D = 0.
  • Угол между плоскостями = угол между нормалями.
  • Площадь сечения: Sсеч = Sпр / cos φ.
