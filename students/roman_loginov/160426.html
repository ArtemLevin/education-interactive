Параметр и модуль: когда уравнение имеет ровно один корень

Разбираем задачу графо-аналитическим методом: аккуратно упрощаем уравнение, переходим к зависимости a = f(x), строим график на нужном интервале и считаем точки пересечения с горизонтальной прямой.

Главная мысль. После преобразований задача сводится к вопросу: при каких значениях параметра a горизонтальная прямая пересекает график функции a = 2|x| - x ровно один раз на интервале (-1;1)?
Идея решения. Сначала превратим исходное уравнение в квадрат выражения, затем получим связь a = 2|x| - x. После этого число корней исходного уравнения будет равно числу пересечений графика этой функции с горизонтальной прямой a = const на интервале -1 < x < 1.
1

Что дано и что нужно найти

Нужно найти все значения параметра a, при которых уравнение

(2|x| + x + a)2 = 8x2 + 2(x + a)2

имеет единственный корень на интервале (-1;1).

То есть нас интересует не просто существование корня, а точное число корней: ровно один.

2

Преобразуем уравнение

Перенесём всё в одну сторону и заметим полный квадрат:

(2|x| + x + a)2 = 4x2 + 4|x|(x+a) + (x+a)2

4x2 + 4|x|(x+a) + (x+a)2 = 8x2 + 2(x+a)2

(2|x| - x - a)2 = 0

Отсюда следует:

2|x| - x - a = 0,   то есть   a = 2|x| - x.
3

Разбираем модуль по случаям

Случай 1: x ≥ 0

Тогда |x| = x, значит

a = 2x - x = x

На этом участке получается правая ветвь: a = x.

Случай 2: x < 0

Тогда |x| = -x, значит

a = 2(-x) - x = -3x

На этом участке получается левая ветвь: a = -3x.

4

Смотрим только на интервал (-1;1)

Это очень важно. Мы не исследуем весь график на всей оси. Нас интересуют только точки с координатами -1 < x < 1.

Поэтому:

  • ветвь a = x работает при 0 ≤ x < 1 и даёт значения a ∈ [0;1);
  • ветвь a = -3x работает при -1 < x < 0 и даёт значения a ∈ (0;3).
5

Геометрический смысл

Для фиксированного значения a нужно провести горизонтальную прямую a = const.

Число корней исходного уравнения равно числу точек пересечения этой прямой с графиком функции a = 2|x| - x на заданном интервале.

6

Разбираем значения параметра

Если a < 0
Пересечений нет.
Если a = 0
Обе ветви сходятся в точке (0,0). Это одна точка, значит один корень.
Если 0 < a < 1
Горизонтальная прямая пересекает обе ветви. Получаются два корня.
Если a = 1
Правая ветвь дала бы x = 1, но это не входит в интервал. Левая даёт один допустимый корень.
Если 1 < a < 3
Есть только одно пересечение с левой ветвью.
Если a ≥ 3
При a = 3 получаем x = -1, это край интервала и он не входит. При больших a пересечений нет.
7

Ответ и проверка

a = 0   или   1 ≤ a < 3

Итак, все подходящие значения параметра:

a ∈ {0} ∪ [1;3)

Быстрая проверка: при a = 2 есть один корень x = -2/3; при a = 1/2 корней уже два.

Ошибка 1. Считать, что при a = 0 есть два корня. На самом деле ветви сходятся в одной и той же точке x = 0.
Ошибка 2. Включать в ответ a = 3. Тогда получается x = -1, а это не принадлежит открытому интервалу (-1;1).
Ошибка 3. Не разбирать модуль. Без перехода к двум ветвям a=x и a=-3x невозможно корректно посчитать число решений.
Ошибка 4. Исследовать график на всей оси, а не только на нужном промежутке.
1. Что означает горизонтальная прямая a = const?
2. Почему a = 1 входит в ответ?
3. Что происходит при 0 < a < 1?

Ключевая формула

(2|x| - x - a)2 = 0   ⇔   a = 2|x| - x

Ветви графика

a = x,   x ≥ 0;    a = -3x,   x < 0

Решающая идея

Считаем пересечения с горизонтальными прямыми и обязательно учитываем открытость интервала.

Итоговый ответ

a ∈ {0} ∪ [1;3)
