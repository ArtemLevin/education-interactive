математика. наглядно

задача 2

Найти все значения параметра a, при которых система
|x| + |a| ≤ 4,
x² + 8x < 16a + 48 имеет хотя бы одно решение на отрезке [-1;0].

главная идея

Сначала используем условие x ∈ [-1;0]. На этом отрезке |x| = -x. Затем сравниваем коридор по a с нижней параболической границей.

идея решения

На отрезке [-1;0] имеем |x| = -x. Тогда первое неравенство превращается в двойной коридор -(4+x) ≤ a ≤ 4+x, а второе даёт область выше параболы a > (x² + 8x - 48)/16.

разбор по шагам

дано

Система и ограничение x ∈ [-1;0].

используем отрезок

На [-1;0] верно |x| = -x.

первое неравенство

-x + |a| ≤ 4 ⇔ |a| ≤ 4+x ⇔ -(4+x) ≤ a ≤ 4+x.

второе неравенство

x² + 8x < 16a + 48 ⇔ a > (x² + 8x - 48)/16.

сравниваем нижние границы

Из равенства (x² + 8x - 48)/16 = -4 - x получаем x² + 24x + 16 = 0, отсюда x = -12 ± 8√2. На отрезке лежит только x₀ = -12 + 8√2.

левая граница по a

a₀ = -(4+x₀) = 8 - 8√2. Из-за строгого знака получаем a > 8 - 8√2.

правая граница

Из a ≤ 4 + x наибольшее значение при x = 0, то есть a = 4. Оно подходит.

вывод

a ∈ (8 - 8√2; 4].

проверка ответа

правая граница входит

При a = 4, x = 0 выполняются оба условия.

левая граница не входит

При a = 8 - 8√2 получается равенство во втором условии, а нужен строгий знак.

типичные ошибки

  • Раскрыть модуль без использования условия x ∈ [-1;0].
  • Потерять строгий знак.
  • Неверно сравнить нижние границы.

самопроверка

1. Почему |x| = -x на этом отрезке?

Показать ответ

Потому что все точки отрезка неположительны.

2. Почему верхняя граница замкнутая, а нижняя открытая?

Показать ответ

Верхняя идёт из нестрогого условия, нижняя — из строгого.
