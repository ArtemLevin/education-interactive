Ровно один корень на отрезке: графо-аналитический разбор

Разбираем уравнение (x² − 10x + a²) / √((a − x + 8)(a + x − 3)) = 0 на отрезке [2; 6]. Слева — подробный пошаговый разбор, справа — интерактивная модель в плоскости (x, a) с зумом, панорамированием, подсветкой шагов и генератором похожих задач.

Тема
Уравнение с параметром
Метод
Графо-аналитический
Ключевая идея
Числитель = 0, ОДЗ > 0
Ответ
(-2√6; -(3+√41)/2) ∪ [4; 2√6) ∪ {-5, 5}

Структура решения

Идея решения. Корень дроби появляется только там, где числитель равен нулю, а знаменатель определён и не равен нулю. Поэтому мы строим окружность из числителя, две прямые из границ ОДЗ, ограничиваемся полосой 2 ≤ x ≤ 6 и по горизонталям a = const считаем число допустимых точек.
1 Что дано и что ищем
Нужно найти все значения параметра a, при которых уравнение имеет ровно один корень по x на отрезке [2; 6].
2 Числитель равен нулю
Из x² − 10x + a² = 0 получаем (x − 5)² + a² = 25. Это окружность с центром (5, 0) и радиусом 5.
3 Ограничения из знаменателя
Так как в знаменателе корень, нужно строгое неравенство: (a − x + 8)(a + x − 3) > 0. Его границы — прямые a = x − 8 и a = −x + 3.
4 Полоса по x
Нас интересуют только точки внутри вертикальной полосы 2 ≤ x ≤ 6.
5 Формулы для корней числителя
На окружности x = 5 ± √(25 − a²). Это даёт левый корень x₁ = 5 − √(25 − a²) и правый корень x₂ = 5 + √(25 − a²).
6 Когда корень ровно один
Нужно отдельно проверить, когда в отрезке остаётся только x₁, когда остаётся только x₂, и не забыть про особые случаи a = ±5, когда обе формулы сливаются в один корень x = 5.
7 Итог
Ответ: a ∈ (−2√6; −(3+√41)/2) ∪ [4; 2√6) ∪ {−5, 5}.

Типичные ошибки

Ошибка 1. Считать, что из знаменателя достаточно ≥ 0. Здесь нужно именно > 0, потому что знаменатель не может равняться нулю.
Ошибка 2. Оставить все точки окружности. На самом деле часть окружности выколота границами ОДЗ.
Ошибка 3. Не учитывать отрезок [2; 6]. Корни вне этой полосы не считаются.
Ошибка 4. Потерять случаи a = ±5. В этих точках окружность даёт один-единственный корень x = 5.

Почему особые случаи важны

Когда a = 5 или a = −5, уравнение числителя превращается в (x − 5)² = 0. Это один корень, а не два, и он допустим по ОДЗ.
