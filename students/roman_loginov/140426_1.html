Уравнения с параметром

Дробь с радикалом: поиск параметра для одного корня на отрезке.

Условие задачи

Найдите все значения a, при каждом из которых уравнение

x2 − 10x + a2 √(a−x+8)(a+x−3) = 0

имеет ровно один корень на отрезке [2; 6].

1
Числитель равен нулю

Дробь равна нулю, когда числитель = 0, а знаменатель ≠ 0.

x2 − 10x + a2 = 0

Дискриминант: D = 100 − 4a2 = 4(25 − a2).

Чтобы были корни, нужно D ≥ 0 ⇒ a ∈ [−5; 5].

Корни: x1,2 = 5 ± √(25 − a2).

Обозначим x1 ≤ 5, x2 ≥ 5.

2
Область допустимых значений (ОДЗ)

Знаменатель должен быть > 0:

(a − x + 8)(a + x − 3) > 0.

Преобразуем в стандартный вид (по x):

(x − (a + 8))(x − (3 − a)) < 0.

Корни знаменателя: xA = a + 8, xB = 3 − a.

Интервал ОДЗ лежит строго между этими точками.

3
Анализ отрезка [2; 6]

Нам нужно, чтобы ровно один из корней x1, x2 попал в пересечение отрезка [2; 6] и ОДЗ.

Ключевые точки:

  • Центр параболы x=5. Отрезок [2; 6] симметричен относительно 4, но корни симметричны относительно 5.
  • x1 ∈ [2; 5], x2 ∈ [5; 8] (при D≥0).

Используем визуализацию справа для поиска критических значений a.

4
Решение и Ответ

Проведя анализ (см. графику), получаем решение:

Ответ:

[−5; (-3−√41)/2) ∪ [(-2−√46)/2; −4] ∪ {4; 5}

Типичные ошибки

1. Забываем ОДЗ

Самая частая ошибка — решать числитель и проверять корни только по отрезку [2; 6], игнорируя, что знаменатель должен быть строгo больше нуля. Корень может попасть в [2; 6], но "выколоться" знаменателем.

2. Неправильный порядок корней ОДЗ

При a = −3 точки a+8 и 3−a равны. При переходе через −2.5 они меняются местами. Нужно внимательно следить, какая точка левая, а какая правая.

3. Граничные точки

Неясно, включаются ли концы отрезка [2; 6]. В данном случае, если корень равен 2 или 6, он "считается", если он не выколот ОДЗ.
