Уравнения с параметром

Графо-аналитический метод, ОДЗ и отбор корней на отрезке
Лёвин Артём Александрович · ЕГЭ, 11 класс

Введение

В задачах этого типа недостаточно просто приравнять числитель к нулю. Нужно отдельно проверить область допустимых значений, потому что знаменатель содержит корень, а затем оставить только те корни, которые лежат на заданном отрезке по x.

Наиболее надёжный способ — перейти к плоскости (x; a), построить график числителя и границы ОДЗ, а затем по горизонталям a = const подсчитать число допустимых точек. Используйте интерактивную панель справа, чтобы наглядно увидеть, как горизонталь a = const пересекает окружность и область ОДЗ.

Тема 1. Теория

Если дробь равна нулю:

\[ \frac{F(x,a)}{G(x,a)} = 0 \quad\Longleftrightarrow\quad F(x,a)=0 \;\text{и}\; G(x,a)\ne 0. \]

В задачах с корнем в знаменателе это означает:

\[ \sqrt{H(x,a)}\ \text{определён и не равен нулю} \quad\Longleftrightarrow\quad H(x,a) > 0. \]
⚠ Типичная ошибка: заменить условие H(x,a) > 0 на H(x,a) ≥ 0, забыв, что знаменатель не может быть равен нулю. Точки на границе ОДЗ не являются допустимыми корнями.

Рабочий алгоритм

  1. Приравнять числитель к нулю.
  2. Перейти к плоскости (x; a) и построить полученный график.
  3. Выписать ОДЗ из знаменателя.
  4. Построить граничные линии ОДЗ и определить допустимую область.
  5. Учесть ограничение на x, то есть оставить только нужную вертикальную полосу.
  6. Для каждого уровня a = const подсчитать число допустимых точек.

Задача 1

Условие

Найдите все значения a, при каждом из которых уравнение

\[ \frac{x^2 - 10x + a^2}{\sqrt{(a-x+8)(a+x-3)}} = 0 \]

имеет ровно один корень на отрезке [2; 6].

Решение

Шаг 1. Приравниваем числитель к нулю

\[ x^2 - 10x + a^2 = 0 \quad\Longleftrightarrow\quad (x-5)^2 + a^2 = 25. \]

В плоскости (x; a) это окружность с центром (5; 0) и радиусом 5. Она отображена на графике справа (серая / бирюзовая линия).

Шаг 2. Выписываем ОДЗ

Знаменатель содержит корень, поэтому подкоренное выражение должно быть строго больше нуля:

\[ (a - x + 8)(a + x - 3) > 0. \]

Граничные прямые:

\[ a = x - 8 \qquad\text{и}\qquad a = -x + 3. \]

Допустимы точки, где оба множителя одного знака:

\[ a > \max(x-8,\,-x+3) \quad\text{или}\quad a < \min(x-8,\,-x+3). \]

На графике эти прямые показаны пунктиром. Закрашенные области — допустимые зоны ОДЗ.

Шаг 3. Учитываем отрезок x ∈ [2; 6]

Работаем только в вертикальной полосе 2 ≤ x ≤ 6. На графике она выделена полупрозрачным фоном.

Шаг 4. Графическая схема

Перемещайте слайдер на панели справа, чтобы увидеть, как горизонталь a = const пересекает окружность и области ОДЗ. Точки пересечения, попавшие в допустимую зону и вертикальную полосу, — это кандидаты в корни.

💡 Попробуйте: установите a = 4 — увидите один корень. Установите a = 5 — касание в одной точке x = 5. Установите a = -5 — аналогично.

Шаг 5. Алгебраическая запись корней

Из (x−5)² + a² = 25 получаем:

\[ x = 5 \pm \sqrt{25 - a^2}. \]

Обозначим t = √(25 − a²), тогда x₁ = 5 − t, x₂ = 5 + t.

Шаг 6. Анализ левого корня x₁ = 5 − t

Условие x₁ ∈ [2; 6] даёт t ≤ 3, то есть |a| ≥ 4.

Проверка ОДЗ подстановкой x₁ = 5 − t в (a − x₁ + 8)(a + x₁ − 3) > 0 приводит к:

\[ (a + t + 3)(a - t + 2) > 0. \]

Отсюда два набора уровней:

\[ a < -\frac{3+\sqrt{41}}{2} \quad\text{или}\quad a > \frac{\sqrt{46}}{2} - 1. \]

С учётом |a| ≥ 4:

\[ a \in \left[-5;\, -\frac{3+\sqrt{41}}{2}\right) \cup [4;\, 5]. \]

Шаг 7. Анализ правого корня x₂ = 5 + t

Условие x₂ ∈ [2; 6] даёт t ≤ 1, то есть |a| ≥ 2√6.

Проверка ОДЗ подстановкой x₂ = 5 + t:

\[ (a - t + 3)(a + t + 2) > 0. \]

Отсюда:

\[ a < -\frac{\sqrt{46}}{2} - 1 \quad\text{или}\quad a > \frac{\sqrt{41}}{2} - \frac{3}{2}. \]

С учётом |a| ≥ 2√6:

\[ a \in [-5;\, -2\sqrt{6}] \cup [2\sqrt{6};\, 5]. \]

Шаг 8. Подсчёт числа корней

Нам нужно, чтобы допустимый корень был ровно один.

Диапазон aКорниПодходит?

Шаг 9. Ответ

\[ a \in \left(-2\sqrt{6};\, -\frac{3+\sqrt{41}}{2}\right) \cup [4;\, 2\sqrt{6}) \cup \{-5,\, 5\} \]

Числовая прямая ответа показана в нижней части боковой панели.

Блиц-опрос

  1. Какой график задаёт уравнение x² − 10x + a² = 0 в плоскости (x; a)?
  2. Почему точка на границе ОДЗ не может быть корнем исходного уравнения?
  3. Почему при a = 5 получается один корень, а не два?
