Параметр как вторая координата: решаем неравенство в плоскости x–a

Разбираем вторую задачу из занятия: строим граничные линии, отмечаем допустимые области, включаем «лазерную указку» a = const и смотрим, когда множество решений по x является именно лучом, а не «лучом плюс кусок».

Тема: задачи с параметром
Фокус: рациональное неравенство
Ключевая идея: сечение горизонтальной прямой
Ответ: только a = 1

Задача

Найдите все положительные значения параметра a, при каждом из которых множество решений неравенства

(x − 2) / (a x² − (a² + 1)x + a) ≥ 0

является некоторым лучом.
Коротко: в координатах (x,a) мы не ищем сразу все решения аналитически. Мы строим границы, определяем подходящие области, а потом для каждого фиксированного a читаем по горизонтальному сечению, какой набор x получается.

Что дано и что нужно найти

Что дано

  • рациональное неравенство;
  • числитель: x − 2;
  • знаменатель: a x² − (a² + 1)x + a;
  • рассматриваются только a > 0.

Что нужно понять

  • какие области плоскости дают знак ≥ 0;
  • как выглядит множество решений по x при фиксированном a;
  • при каком a получается именно луч.

Идея решения

Шаг 1. Строим граничную линию числителя: x − 2 = 0, то есть x = 2. Это вертикальная прямая в плоскости (x,a).
Шаг 2. Строим граничные линии знаменателя: a x² − (a² + 1)x + a = 0.
Шаг 3. Разбиваем плоскость на области и подстановкой определяем, где исходная дробь неотрицательна.
Шаг 4. Для каждого фиксированного a проводим горизонтальную прямую и читаем по ней множество решений по x.

Пошаговое решение

1. Граница числителя

x − 2 = 0 ⟹ x = 2

Это вертикальная прямая. Она важна потому, что в точке x = 2 числитель равен нулю.

2. Граница знаменателя

a x² − (a² + 1)x + a = 0

Рассматриваем это как квадратное уравнение относительно x.

D = (a² + 1)² − 4a² = a⁴ − 2a² + 1 = (a² − 1)²
x₁ = ((a² + 1) + (a² − 1)) / (2a) = a,    x₂ = ((a² + 1) − (a² − 1)) / (2a) = 1/a
Ключевой вывод: граничные линии знаменателя — это a = x и a = 1/x.

3. Очень важная деталь: особый случай a = 1

При a = 1 две линии знаменателя совпадают, потому что a = x и a = 1/x пересекаются в точке (1,1), а сам знаменатель превращается в квадрат:

x² − 2x + 1 = (x − 1)²

4. Аналитическая форма для знаков

a x² − (a² + 1)x + a = a(x − a)(x − 1/a),  при a > 0

Так как a > 0, знак знаменателя определяется только произведением (x − a)(x − 1/a).

Снаружи от корней a и 1/a знаменатель положителен.
Между ними знаменатель отрицателен.
Числитель x − 2 отрицателен слева от 2 и положителен справа от 2.

5. Почему почти всегда получается не луч

Если горизонтальная прямая a = const пересекает области так, что появляются две отдельные части решений по x, то это уже не луч. Именно это и происходит почти при всех положительных a.

Тонкое место. Если в множестве решений есть «отрезок или интервал слева» и отдельно «луч справа», то условие задачи не выполнено. Нужен ровно один луч.

6. Проверяем особый кандидат a = 1

(x − 2) / (x − 1)² ≥ 0

Знаменатель положителен при всех x ≠ 1, поэтому знак дроби совпадает со знаком x − 2. Получаем:

x ≥ 2

То есть множество решений — это луч [2; +∞).

Ответ: единственное положительное значение параметра, при котором множество решений является лучом, — это a = 1.

Отдельный блок: как мыслить через «лазерную указку»

Вы фиксируете a, то есть проводите горизонтальную прямую.
Всё, что лежит на этой прямой внутри подходящей области, даёт решения по x.
Если на линии получился только правый кусок вида [c; +∞) или (c; +∞), это луч.
Если появился ещё один кусок отдельно — задача уже не подходит.

Типичные ошибки

  • Не проверить, что a = 1 — особый случай, когда корни знаменателя совпали.
  • Сократить на x − 2 в случае, когда он входит в знаменатель, и забыть про выколотую точку.
  • Считать, что «луч плюс отдельный интервал» — это всё ещё луч.
  • Подставлять точки не в исходное неравенство, а только в знаменатель.
  • Смотреть только на график, но не проверять ОДЗ: точки знаменателя никогда не входят в решение.

Самопроверка

  • Я умею построить в плоскости x–a линии x = 2, a = x, a = 1/x.
  • Я понимаю, почему знак читается по областям.
  • Я понимаю, почему при большинстве a > 0 получается не один луч, а несколько кусков.
  • Я отдельно проверяю случай a = 1.
  • Я могу объяснить, почему правильный ответ именно 1.
