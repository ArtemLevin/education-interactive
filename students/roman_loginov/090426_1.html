Входные настройки

Ограничения: без внешних библиотек, без LaTeX-рендереров, только HTML/CSS/JS.

Разбор задачи

Найти все a, при которых система имеет хотя бы одно решение:

{ (a + 7x + 4)(a − 2x + 4) ≤ 0;   a + 3x ≥ x² }

Наведи на шаг — соответствующие объекты подсветятся справа.

Идея решения

Пересечь множество решений квадратичного неравенства x² − 3x − a ≤ 0 с множеством решений произведения линейных множителей ≤ 0. Для a ≥ −9/4 второй блок даёт отрезок [L;U], а первый — внешность интервала (x₁;x₂). Ищем, когда [L;U] не целиком внутри (x₁;x₂).

Типичные ошибки

  • Путают правило: log/произведение? (здесь его нет) — нужно только знаки множителей.
  • Забывают условие существования корней x² − 3x − a ≤ 0: a ≥ −9/4.
  • Теряют особую точку a = 10.

Самопроверка

  • Проверь a=4 (должно быть решение).
  • Проверь a=5 (решений нет).
  • Проверь a=10 (решение есть по границе).

Краткий итог / takeaway

Ответ: a ∈ [−9/4; 4] ∪ {10}.

Генератор похожих задач

Сгенерируем систему вида (a + px + c)(a − qx + c) ≤ 0 и a + rx ≥ x².

Печатный конспект

1) Что дано: система из произведения линейных множителей и параболы по x. 2) Что найти: все a, где есть хотя бы одно x. 3) Ключ: неравенство x²−3x−a≤0 задаёт отрезок. 4) Проверка: границы a=−9/4,4 и особая точка a=10.
