задача 1

Найти все значения параметра a, при которых система
(a + 7x + 4)(a - 2x + 4) ≤ 0,
a + 3x ≥ x² имеет хотя бы одно решение.

что здесь главное

Переходим к плоскости (x,a). Первое условие задаёт полосу между двумя прямыми, второе — область над параболой. Дальше ищем, при каких уровнях a = const общая область существует.

идея решения

Рассматриваем параметр a как вторую координату.

  • (a+7x+4)(a-2x+4) ≤ 0 означает, что a лежит между прямыми a = -7x - 4 и a = 2x - 4.
  • a + 3x ≥ x² означает a ≥ x² - 3x, то есть область над параболой.

Нужно понять, где парабола не выше верхней границы полосы.

разбор по шагам

дано

Система:
(a + 7x + 4)(a - 2x + 4) ≤ 0,
a + 3x ≥ x².

смысл первого условия

Корни по a: a = -7x - 4 и a = 2x - 4. Значит, параметр находится между этими значениями.

смысл второго условия

a + 3x ≥ x² ⇔ a ≥ x² - 3x. Значит, точка лежит над параболой.

какая прямая сверху

-7x - 4 ≥ 2x - 4 ⇔ x ≤ 0. Значит, сверху: -7x - 4 при x ≤ 0 и 2x - 4 при x ≥ 0.

ищем допустимые x

Для x ≤ 0: x² - 3x ≤ -7x - 4 ⇔ (x+2)² ≤ 0, значит x = -2.

Для x ≥ 0: x² - 3x ≤ 2x - 4 ⇔ (x-1)(x-4) ≤ 0, значит x ∈ [1;4].

переход к параметру a

При x = -2 получаем a = 10.

При x ∈ [1;4] параметр пробегает значения от минимума параболы -9/4 до максимума прямой 4.

вывод

a ∈ [-9/4; 4] ∪ {10}.

проверка ответа

точка внутри ответа

При a = 0, x = 2: (18)·0 = 0 и 6 ≥ 4. Решение есть.

точка вне ответа

При a = 5 горизонтальный уровень не пересекает допустимую область, значит решений нет.

типичные ошибки

  • Требовать выполнение для всех x, а не для хотя бы одного.
  • Забыть разбиение по случаям x ≤ 0 и x ≥ 0.
  • Неверно искать минимум x² - 3x на [1;4].

самопроверка

1. Почему значение a = 10 идёт отдельно?

Показать ответ

Оно возникает только из единственной точки x = -2.

2. Где достигается минимум x² - 3x на [1;4]?

Показать ответ

При x = 3/2.
