Короткий учебный маршрут: ограничения → преобразования → замена → метод интервалов → обратная замена → проверка. Для кредитов: таблица по годам и уравнения по смыслу.

Главное правило:

logab существует, если a>0, a≠1, b>0.

Если логарифм в знаменателе: logab≠0.

Сначала ограничения. Потом любые красивые преобразования.

1. ОДЗ: входной контроль

Для логарифма

logab: основание положительно и не равно 1, аргумент положителен.

a>0, a≠1, b>0

Для знаменателя

Если логарифм стоит внизу дроби, он не может быть нулём.

logab≠0 ⇔ b≠1

Микро-пример

Для 1 / log2x получаем: x>0 и x≠1. Точка x=1 запрещена, потому что знаменатель станет нулём.

2. Алгоритм решения

Записать ОДЗ: основания, аргументы, знаменатели.
Привести логарифмы к одному виду: чаще всего к log2x.
Сделать замену: t = log2x.
Решить рациональное или квадратное неравенство по t методом интервалов.
Записать промежутки через систему и совокупность.
Вернуться к x и обязательно сверить ответ с ОДЗ.

4. Ошибки, которые снимают баллы

Домножение

Нельзя домножать неравенство на переменное выражение без знания его знака.

Знаменатель

Критическая точка знаменателя не входит в ответ, даже если метод интервалов даёт нужный знак.

Система

Отрезок задаётся двумя условиями одновременно: это система.

Совокупность

Объединение разных промежутков: это «или», то есть совокупность.

7. Мини-квиз

Печатная памятка

  1. ОДЗ пишем до преобразований.
  2. Знаменатель не равен нулю.
  3. На переменное выражение без знака не домножаем.
  4. Дробные неравенства решаем методом интервалов.
  5. В кредитах строим таблицу и проверяем остаток долга.

Интерактивная модель

Модель для неравенства:

((t−a)(t+b))/t ≤ 0

Если долг не меняется, платёж равен процентам.

Платёж: 1,65 млн

  • Замена: повторяющийся логарифм обозначаем через t.
  • Интервалы: нули числителя и знаменателя на ось.
  • Кредит: последний остаток равен 0.
