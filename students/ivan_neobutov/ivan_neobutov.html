Теория и схема решения

Что обязательно для логарифма

logab определён только если
a > 0, a ≠ 1, b > 0.
Самая частая ошибка: проверить только аргумент и забыть про условие a ≠ 1.

Знак логарифма

Если a > 1, то logab > 0 при b > 1.
Если 0 < a < 1, то logab < 0 при b > 1.
Во второй задаче аргумент всегда больше 1 на ОДЗ, поэтому решает именно знак основания.

Схема решения неравенства

Для задачи 1

  1. Проверить, что x > 0.
  2. Сделать замену t = lg x.
  3. Разложить многочлен на множители.
  4. Решить неравенство по t методом интервалов.
  5. Вернуться к x.

Для задачи 2

  1. Выписать ОДЗ логарифма.
  2. Понять, что x² − 6x + 10 = (x − 3)² + 1.
  3. Сравнить аргумент с 1.
  4. Разбить ОДЗ по знаку основания 2x − 5.
  5. Перемножить знаки двух множителей.
задача 9

Решение 1: замена t = lg x

lg⁴x − 4lg³x + 5lg²x − 2lgx ≥ 0
Идея решения. Здесь все степени стоят у одного и того же выражения lg x, значит естественно перейти к новой переменной и получить обычное алгебраическое неравенство.
1

ОДЗ

смысл условия

Логарифм lg x существует только при x > 0.

ОДЗ: x > 0
2

Замена переменной

перевод к алгебре

Положим t = lg x. Тогда исходное неравенство превратится в

t⁴ − 4t³ + 5t² − 2t ≥ 0
3

Разложение

ключевая алгебра
t⁴ − 4t³ + 5t² − 2t = t(t³ − 4t² + 5t − 2)

Проверяем корень t = 1: 1 − 4 + 5 − 2 = 0.

t³ − 4t² + 5t − 2 = (t − 1)(t² − 3t + 2) = (t − 1)²(t − 2)
Итак: t(t − 1)²(t − 2) ≥ 0
4

Знаки на оси t

метод интервалов

Множитель (t − 1)² не меняет знак, потому что квадрат неотрицателен. Он только даёт нуль в точке t = 1.

t ∈ (−∞, 0] ∪ {1} ∪ [2, +∞)
5

Возврат к x

финальный перевод
lg x ≤ 0 ⟹ 0 < x ≤ 1
lg x = 1 ⟹ x = 10
lg x ≥ 2 ⟹ x ≥ 100
Ответ: x ∈ (0, 1] ∪ {10} ∪ [100, +∞)

Типичные ошибки

  • Потерять условие x > 0.
  • Считать, что в точке t = 1 знак меняется. Нет: корень чётной кратности знак не меняет.
  • После решения по t забыть вернуться к x.
задача 11

Решение 2: анализ знака логарифма

(5x − 13) · log2x−5(x² − 6x + 10) ≥ 0
Идея решения. Здесь невыгодно раскрывать логарифм. Намного умнее понять, когда сам логарифм положителен или отрицателен.
1

ОДЗ

основание и аргумент
2x − 5 > 0, 2x − 5 ≠ 1, x² − 6x + 10 > 0
x² − 6x + 10 = (x − 3)² + 1 > 0 всегда
Итак: x > 5/2, x ≠ 3
2

Сравнение аргумента с 1

самая важная мысль
x² − 6x + 10 = (x − 3)² + 1 ≥ 1

Равенство возможно только при x = 3, но эта точка исключена из ОДЗ. Значит на ОДЗ всегда

x² − 6x + 10 > 1
3

Знак логарифма

основа метода

Если 5/2 < x < 3, то 0 < 2x − 5 < 1, а логарифм числа, большего 1, при таком основании отрицателен.

Если x > 3, то 2x − 5 > 1, и логарифм числа, большего 1, становится положительным.

4

Знак множителя 5x − 13

второй множитель
5x − 13 = 0 ⟹ x = 13/5

Точка 13/5 = 2.6 лежит между 5/2 и 3.

5

Итог по интервалам

перемножаем знаки
(5/2, 13/5): (−)·(−) = (+)
x = 13/5: 0
(13/5, 3): (+)·(−) = (−)
(3, +∞): (+)·(+) = (+)
Ответ: x ∈ (5/2, 13/5] ∪ (3, +∞)

Типичные ошибки

  • Забыть, что при основании 0 < a < 1 знак логарифма устроен наоборот.
  • Не исключить точку x = 3, где основание равно 1.
  • Сравнивать аргумент логарифма с нулём, а нужно — с единицей.

Тренировка и самопроверка

Тренировка 1

lg⁴x − 2lg³x − lg²x + 2lgx ≤ 0

Подсказка: сделайте замену t = lg x и разложите многочлен на множители.

Показать ответ
x ∈ [1/10, 1] ∪ [10, 100]

Тренировка 2

(4x − 9) · log2x−3(x² − 4x + 5) ≤ 0

Подсказка: x² − 4x + 5 = (x − 2)² + 1, то есть аргумент снова больше 1 на ОДЗ.

Показать ответ
x ∈ (2, 9/4]
