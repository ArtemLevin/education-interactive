Построение графика функции с модулем

Исследование пересечения графика y = f(x) с прямой y = kx. Пошаговый разбор с интерактивной визуализацией.

Задача ОГЭ / 9–11 класс — Функции с модулем и параметром
1 Что дано и что нужно найти ▼

Дана функция:

y = 3,5|x| − 1|x| − 3,5x²

Нужно:

  1. Построить график функции.
  2. Определить, при каких значениях k прямая y = kx не имеет с графиком общих точек.
💡 Ключевая идея: раскрыть модуль по случаям, упростить дробь, затем решить уравнение пересечения.
2 Область определения (ОДЗ) ▼

Знаменатель не должен обращаться в ноль:

|x| − 3,5x² ≠ 0

Выносим |x| за скобку (учитывая x² = |x|²):

|x| · (1 − 3,5|x|) ≠ 0

Получаем два условия:

① |x| ≠ 0  ⟹  x ≠ 0
② 1 − 3,5|x| ≠ 0  ⟹  |x| ≠ 13,5 = 27  ⟹  x ≠ ±27
ОДЗ: x ∈ (−∞; −27) ∪ (−27; 0) ∪ (0; 27) ∪ (27; +∞)
3 Раскрываем модуль и упрощаем ▼
Случай x > 0 (при этом x ≠ 2/7):
|x| = x
y = 3,5x − 1x − 3,5x² = 3,5x − 1x(1 − 3,5x) = 3,5x − 1−x(3,5x − 1) = −1x
Случай x < 0 (при этом x ≠ −2/7):
|x| = −x
y = −3,5x − 1−x − 3,5x² = −(3,5x + 1)−x(1 + 3,5x) = 3,5x + 1x(1 + 3,5x) = 1x

Итоговая кусочная запись:

y = ⎧1x, x < 0, x ≠ −27
⎨
⎩−1x, x > 0, x ≠ 27
4 Исследуем пересечение с прямой y = kx ▼

При x > 0: −1/x = kx

kx² = −1
  • При k ≥ 0: уравнение не имеет решений (левая часть ≥ 0, правая < 0)
  • При k < 0: x = √(−1/k) — одно решение (если оно не равно выколотой точке)

При x < 0: 1/x = kx

kx² = 1
  • При k ≤ 0: уравнение не имеет решений (левая часть ≤ 0, правая > 0)
  • При k > 0: x = −√(1/k) — одно решение (если не равно выколотой точке)
5 Проверяем выколотые точки ▼

Если корень уравнения попадает в выколотую точку, то решения нет.

x = 2/7 (выколота справа):
Подставляем в kx² = −1:
k · (2/7)² = −1 ⟹ k · 4/49 = −1 ⟹ k = −49/4 = −12,25
x = −2/7 (выколота слева):
Подставляем в kx² = 1:
k · (−2/7)² = 1 ⟹ k · 4/49 = 1 ⟹ k = 49/4 = 12,25

При этих значениях k корень «пропадает» — решений нет.

6 Объединяем и получаем ответ ▼

Прямая y = kx не имеет общих точек, если нет решений ни при x > 0, ни при x < 0.

Ответ

k ∈ { −49/4;  0;  49/4 }
или k ∈ { −12,25;  0;  12,25 }

Типичные ошибки

  • Забыть ОДЗ. Не исключить точки x = 0 и x = ±2/7. Без этого ответ будет неверным — «лишние» пересечения в выколотых точках.
  • Неправильное раскрытие модуля. При x < 0 должно быть |x| = −x. Ошибка в знаке ведёт к неправильному упрощению.
  • Путаница со знаками при упрощении. Дробь (3,5x − 1) / (x − 3,5x²) нужно аккуратно преобразовать: вынести минус из знаменателя.
  • Не проверить выколотые точки при подстановке. Даже если уравнение kx² = −1 имеет корень, он может совпасть с x = 2/7 — тогда решения нет.
  • Пропустить случай k = 0. Прямая y = 0 (ось абсцисс) не пересекает график, так как y = ±1/x ≠ 0 при любом x.
  • Смешать условия для x > 0 и x < 0. Для каждой ветви своё уравнение: −1/x = kx и 1/x = kx. Нельзя объединять их в одно.
💡 Запомните: всегда сначала находите ОДЗ, потом упрощаете, потом решаете уравнение пересечения, и только потом проверяете, не попали ли корни в выколотые точки.

Краткий итог

y = 3,5|x| − 1|x| − 3,5x²  ⟹  y = ⎧1x (x<0)  |  −1x (x>0)

После упрощения график состоит из двух ветвей гипербол: y = 1/x при x < 0 и y = −1/x при x > 0, с выколотыми точками при x = ±2/7.

Прямая y = kx не пересекает график, когда:

  • k = 0 — ось абсцисс не пересекает ни одну ветвь (y ≠ 0)
  • k = −49/4 — корень при x > 0 совпадает с выколотой точкой x = 2/7
  • k = 49/4 — корень при x < 0 совпадает с выколотой точкой x = −2/7

Финальный ответ

k ∈ { −12,25;  0;  12,25 }
