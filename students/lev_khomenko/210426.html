Теория

Скалярное произведение

a · b = a₁b₁ + a₂b₂ + a₃b₃
Если результат больше нуля — угол острый. Если меньше нуля — тупой. Если равен нулю — векторы перпендикулярны.

Векторное произведение

a × b = (a₂b₃ − a₃b₂; a₃b₁ − a₁b₃; a₁b₂ − a₂b₁)
Получается новый вектор, перпендикулярный обоим исходным. Очень важно не потерять знак второй координаты.

Уравнение прямой

y = kx + b,    k = (y₂ − y₁)/(x₂ − x₁)
Сначала находим угловой коэффициент k, затем подставляем координаты одной точки и вычисляем b.

Типичные ошибки

  • Перемножают не соответствующие координаты в скалярном произведении.
  • Теряют знак во второй координате векторного произведения.
  • Несогласованно вычитают координаты при поиске k.

Задача 1. Скалярное произведение

Условие. Вычислить a · b, если a = (−1; 2; 1), b = (2; −1; 3).

a · b = (−1)·2 + 2·(−1) + 1·3
Шаг 1
Перемножаем соответствующие координаты: (−1)·2, 2·(−1), 1·3.
Шаг 2
Получаем числа: −2, −2 и 3.
Шаг 3
Складываем: −2 − 2 + 3 = −1.
Шаг 4
Результат отрицателен, значит угол между векторами тупой.
Идея решения: скалярное произведение — это обычная сумма попарных произведений координат.
Ответ: a · b = −1
2

Задача 2. Векторное произведение

Условие. Вычислить a × b, если a = (4; −1; 1), b = (2; −3; 1).

a × b = (a₂b₃ − a₃b₂; a₃b₁ − a₁b₃; a₁b₂ − a₂b₁)
Шаг 1
Первая координата: (−1)·1 − 1·(−3) = −1 + 3 = 2.
Шаг 2
Вторая координата: 1·2 − 4·1 = 2 − 4 = −2.
Шаг 3
Третья координата: 4·(−3) − (−1)·2 = −12 + 2 = −10.
Шаг 4
Получаем вектор (2; −2; −10). Он должен быть перпендикулярен обоим исходным — это хорошая проверка.
Идея решения: здесь не число, а новый вектор. Поэтому мы считаем сразу три координаты по специальной формуле.
Ответ: a × b = (2; −2; −10)
3

Задача 3. Уравнение прямой

Условие. Составить уравнение прямой, проходящей через точки K(1; 1) и L(2; 5), в виде уравнения с угловым коэффициентом.

k = (5 − 1)/(2 − 1) = 4,    y = 4x + b
Шаг 1
Находим угловой коэффициент: k = (5 − 1)/(2 − 1) = 4.
Шаг 2
Записываем общий вид прямой: y = 4x + b.
Шаг 3
Подставляем точку K(1;1): 1 = 4·1 + b.
Шаг 4
Находим b = −3, значит y = 4x − 3.
Шаг 5
Проверяем точку L(2;5): 4·2 − 3 = 5. Всё верно.
Идея решения: вся задача сводится к нахождению двух чисел — наклона k и свободного члена b.
Ответ: y = 4x − 3

Тренировка

T1

Найдите значение m, при котором u = (m; 2; −1) и v = (3; −1; 4) перпендикулярны.

Показать ответ

3m + 2·(−1) + (−1)·4 = 0 → 3m − 6 = 0 → m = 2

T2

Найдите значение m, если у p = (1; m; 2) и q = (3; −1; 4) третья координата вектора p × q равна −4.

Показать ответ

1·(−1) − 3m = −4 → −1 − 3m = −4 → m = 1

Тогда p × q = (6; 2; −4)

T3

Составьте уравнение прямой в виде y = kx + b, если она проходит через точки A(−2;3) и B(4;−9).

Показать ответ

k = (−9 − 3)/(4 − (−2)) = −12/6 = −2

3 = −2·(−2) + b → b = −1

Ответ: y = −2x − 1
