Скалярное произведение: смысл, формулы, длина, угол и перпендикулярность

Это самая важная часть темы: понять, что именно означает формула a · b = |a||b| cos φ, как она связана с проекцией, и как из неё быстро получать длину суммы, длину разности, угол между векторами и признак перпендикулярности.

Идея решения

Шаг 1. Скалярное произведение — это число, которое измеряет, насколько один вектор направлен вдоль другого.
Шаг 2. Формула a · b = |a||b| cos φ показывает: важны только длины и угол между векторами.
Шаг 3. Величина |a| cos φ — это проекция вектора a на направление b.
Шаг 4. Поэтому a · b = |b| · prba. Аналогично можно проектировать b на a.
Шаг 5. Для длин суммы и разности нужно сначала возвести длину в квадрат и раскрыть скобки через скалярное произведение.

a · b = |a| · |b| · cos φ

Если угол острый, косинус положительный, и скалярное произведение положительно. Если угол прямой, косинус равен нулю. Если угол тупой, косинус отрицательный.

a · b = |b| · prba = |a| · prab

Геометрически мы берём «часть» одного вектора вдоль направления другого. Это самый наглядный смысл скалярного произведения.

(x1, y1, z1) · (x2, y2, z2) = x1x2 + y1y2 + z1z2

Когда векторы заданы координатами, это самая удобная форма для вычислений, признака перпендикулярности и поиска угла.

a · b = b · a
(λa) · b = λ(a · b)
a · (b + c) = a · b + a · c
a · a = |a|²
Ловушка. Скалярное произведение — это число, а не вектор.

Пошаговый разбор главного примера

из пособия

Пусть c = 3a − 4b, |a| = 2, |b| = 3, угол между a и b равен 60°. Найти |c|.

|c|² = (3a − 4b) · (3a − 4b) = 9|a|² − 24(a · b) + 16|b|²
a · b = |a||b| cos 60° = 2 · 3 · 1/2 = 3
|c|² = 9·4 − 24·3 + 16·9 = 108, значит |c| = 6√3
1. Ищем не сразу |c|, а сначала квадрат длины: |c|² = c · c.
2. Подставляем c = 3a − 4b:
|c|² = (3a − 4b) · (3a − 4b)
3. Раскрываем скобки:
|c|² = 9(a · a) − 24(a · b) + 16(b · b)
4. Используем a · a = |a|², b · b = |b|² и формулу через угол:
a · b = |a||b| cos 60° = 2 · 3 · 1/2 = 3
5. Подставляем числа:
|c|² = 9·4 − 24·3 + 16·9 = 108
6. Берём корень:
|c| = √108 = 6√3
1. В задачах на длину выражения вида 3a − 4b самый удобный приём — перейти к квадрату длины. Это избавляет от корня на первом шаге и позволяет раскрыть скобки по правилам скалярного произведения.
2. Записываем:
|c|² = c · c = (3a − 4b) · (3a − 4b)
3. Распределяем скалярное произведение по сумме и выносим коэффициенты:
|c|² = 9(a · a) − 12(a · b) − 12(b · a) + 16(b · b)
4. Так как a · b = b · a, средние слагаемые объединяются:
|c|² = 9|a|² − 24(a · b) + 16|b|²
5. Находим смешанное произведение через угол:
a · b = |a||b| cos 60° = 2 · 3 · 1/2 = 3
6. Подставляем длины:
|c|² = 9·2² − 24·3 + 16·3² = 36 − 72 + 144 = 108
7. Окончательно:
|c| = √108 = 6√3
Почему это важно? Тот же шаблон работает для задач на |a+b|, |a-b|, угол между векторами и условие перпендикулярности.

Типичные ошибки

Ошибка 1. Забыть, что cos 120° = -1/2, а не 1/2.
Ошибка 2. Путать a · a с |a|. Верно: a · a = |a|².
Ошибка 3. Искать длину сразу, не переходя к квадрату длины.
Ошибка 4. Писать, что нулевое скалярное произведение всегда означает перпендикулярность, забывая оговорить случай ненулевых векторов.

Связка между текстом и рисунком

a · b = |b| · pr₍b₎a

На рисунке оранжевый отрезок — это проекция a на направление b. Когда угол увеличивается, проекция уменьшается. При 90° она равна нулю, и скалярное произведение тоже равно нулю.

Памятка для печати

|a ± b|² = |a|² ± 2(a · b) + |b|²
a ⟂ b ⇔ a · b = 0
cos φ = (a · b) / (|a||b|)

Этого набора формул достаточно для большинства первых задач на скалярное произведение.
