Исходная задача. Найти все значения параметра a, при которых система

|x - a| + |y| = 2 y = √(4 - x²)

типичные ошибки

Ошибка 1.
Забыть, что по второму уравнению всегда y ≥ 0, и не заменить |y| на y.
Ошибка 2.
Сразу возводить в квадрат и терять геометрию задачи.
Ошибка 3.
Не учитывать, что при a = 0 вершина ломаной (0, 2) общая для двух ветвей, но это одна точка, а не две.
Ошибка 4.
Не выделять критические значения параметра, в которых меняется число решений.

тренировка по мотивам задачи

Задача. Найдите все значения параметра b, при которых система

|x - b| + |y| = 3 y = √(9 - x²)

имеет ровно одно решение.

Это та же конструкция, только радиус полуокружности и высота вершины ломаной равны 3. Поэтому критические значения масштабируются в 1.5 раза относительно основной задачи:

b = -6, b = 3 - 3√2, b = 3√2 - 3, b = 6

Ответ: b ∈ [−6, 3 − 3√2) ∪ (3√2 − 3, 6].
