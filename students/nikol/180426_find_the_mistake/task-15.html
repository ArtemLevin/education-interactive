Задание 15. Отбор корней на промежутке

Эта страница посвящена анализу типичной ошибки. Сначала внимательно читаем ошибочное решение, затем локализуем сбой, объясняем его и восстанавливаем правильную логику.

Исходное задание

2sin x - 1 = 0, x ∈ [0; 2π]

Идея решения

После общего решения обязательно нужен отдельный шаг отбора по промежутку.

Фокус этой страницы: В ответ включено число 13π/6, которое не принадлежит отрезку [0; 2π].

1. Ошибочное решение

Ниже записан правдоподобный, но неверный ход рассуждений. Прочитайте его медленно и попробуйте остановиться в том месте, где логика ломается.

sin x = 1/2
x = π/6 + 2πn или x = 5π/6 + 2πn
На [0; 2π]: π/6, 5π/6, 13π/6

2. Где именно ошибка

Место ошибки
Включение 13π/6 в список корней.
Почему это неверно
13π/6 > 12π/6 = 2π.

3. Пошаговый правильный разбор

1
Сначала решаем простейшее уравнение: sin x = 1/2.
2
Общий вид: x = π/6 + 2πn или x = 5π/6 + 2πn.
3
Сравниваем с границами отрезка: 0 ≤ x ≤ 2π.
4
Подходят только π/6 и 5π/6.

4. Короткий итог

Как правильно
Сравнить каждый кандидат с границами промежутка.
Верный ответ
π/6; 5π/6

5. Типичные ошибки рядом с этой задачей

  • не проверять смысл каждого символа и свойства перед применением;
  • перескакивать через промежуточное объяснение;
  • доверять похожему шаблону, не сверяя знак, область допустимых значений или период;
  • сразу записывать ответ без контрольной проверки.

6. Мини-квиз

Какое число больше: 2π или 13π/6?
