Логарифмические уравнения

Задание 08. ОДЗ и посторонний корень

Эта страница посвящена анализу типичной ошибки. Сначала внимательно читаем ошибочное решение, затем локализуем сбой, объясняем его и восстанавливаем правильную логику.

Исходное задание

log₂(x + 1) + log₂(x - 3) = 3

Идея решения

Логарифмическое уравнение нельзя завершать без отбора по области допустимых значений.

Фокус этой страницы: После решения квадратного уравнения не выполнена проверка ОДЗ.

1. Ошибочное решение

Ниже записан правдоподобный, но неверный ход рассуждений. Прочитайте его медленно и попробуйте остановиться в том месте, где логика ломается.

log₂((x + 1)(x - 3)) = 3
(x + 1)(x - 3) = 8
x² - 2x - 11 = 0
x = 1 ± 2√3
Ответ: 1 ± 2√3

2. Где именно ошибка

Место ошибки
Финальная строка «Ответ: 1 ± 2√3».
Почему это неверно
Один из найденных корней делает аргумент логарифма неположительным.

3. Пошаговый правильный разбор

1
Для исходного уравнения нужны условия: x + 1 > 0 и x - 3 > 0, значит x > 3.
2
После объединения логарифмов получаем (x + 1)(x - 3) = 8.
3
Решая квадратное уравнение, находим x = 1 ± 2√3.
4
Из них условию x > 3 удовлетворяет только x = 1 + 2√3.

4. Короткий итог

Как правильно
Нужно записать ОДЗ в начале и проверить все найденные значения.
Верный ответ
1 + 2√3

5. Типичные ошибки рядом с этой задачей

  • не проверять смысл каждого символа и свойства перед применением;
  • перескакивать через промежуточное объяснение;
  • доверять похожему шаблону, не сверяя знак, область допустимых значений или период;
  • сразу записывать ответ без контрольной проверки.
