Задание 6. Квадратное уравнение после замены

Решите уравнение 2cos²(x/2) + cos(x/2) − 1 = 0.

← Предыдущее Все задания Следующее →

Что дано

2cos²(x/2) + cos(x/2) − 1 = 0
Главная идея
Это уже готовое квадратное уравнение относительно cos(x/2). Остаётся сделать замену и вернуться к углу.
Финальный ответ
x = 2π/3 + 4πn, x = −2π/3 + 4πn, x = 2π + 4πn, n ∈ ℤ.

Пошаговый разбор

1. Замена
Пусть t = cos(x/2), t ∈ [−1,1].
2. Квадратное уравнение
2t² + t − 1 = 0.
3. Корни
t₁ = 1/2, t₂ = −1.
4. Возврат к углу
cos(x/2)=1/2 ⟹ x = ±2π/3 + 4πn. cos(x/2)=−1 ⟹ x = 2π + 4πn.
