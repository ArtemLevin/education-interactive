Тригонометрические уравнения
приведение, двойной угол, группировка

Как сводить сложные уравнения к простейшим, применять формулы понижения степени и аккуратно работать с тригонометрической окружностью.

Разбираем  2cos 2x · sin 4x − cos 4x = 2sin 2x  → отбор корней на окружности
Шаг 1 из 4

Переносим всё в одну сторону, чтобы получить произведение = 0Это ключевая цель любого уравнения:

2cos 2x · sin 4x − cos 4x − 2sin 2x = 0

Замечаем, что углы 4x и 2x связаны. Это подсказка: нужно использовать формулы двойного угла.

Шаг 2 из 4

Применяем тождества:

sin 4x = 2sin 2x cos 2x
cos 4x = 2cos² 2x − 1

Подставляем в уравнение:

2cos 2x · (2sin 2x cos 2x) − (2cos² 2x − 1) − 2sin 2x = 0
4sin 2x cos² 2x − 2cos² 2x + 1 − 2sin 2x = 0
Шаг 3 из 4

Разбиваем на две пары и выносим общие множители:

Первая пара: 2cos² 2x(2sin 2x − 1)
Вторая пара: −1(2sin 2x − 1)

Выносим общую скобку (2sin 2x − 1):

(2cos² 2x − 1)(2sin 2x − 1) = 0

Замечаем: 2cos² 2x − 1 — это формула косинуса двойного угла! Значит, первое выражение равно cos 4x.

Шаг 4 из 4

Произведение равно нулю, когда хотя бы один множитель нуль:

cos 4x = 0   или   sin 2x = 1/2
1) cos 4x = 0 → 4x = π/2 + πk → x = π/8 + πk/4
2) sin 2x = 1/2 → 2x = (−1)ⁿπ/6 + πn → x = (−1)ⁿπ/12 + πn/2
Ответ
x = π/8 + πk/4,   x = (−1)ⁿπ/12 + πn/2,   k,n ∈ ℤ

На окружности точки sin 2x = 1/2 лежат на уровне y = 1/2, а cos 4x = 0 — на осях при удвоенном/учетверённом угле.

❌ Забыли скобки при подстановке

При замене cos 4x = 2cos² 2x − 1 перед скобкой стоит минус. Если не поставить скобки, знак «−1» превратится в «+1», и всё решение пойдёт неверно.

❌ Ошибка в формуле приведения

cos(3π/2 + α) = sin α, а не −sin α. В третьей/четвёртой четвертях косинус положителен. Проверяйте по тригонометрическому кругу, а не наизусть.

❌ Не поделили на коэффициент при x

Если 4x = π/2 + πk, то весь правый ряд нужно разделить на 4. Частая ошибка: делят только π/2, забывая про πk. Итог: x = π/8 + πk/4.

❌ Извлечение корня без формул понижения

Вместо cos² α = (1+cos 2α)/2 часто пишут cos α = ±√a. Это создаёт два сложных уравнения с корнями. Формула понижения степени экономит время и исключает ошибки ОДЗ.

Вопрос 1. Чему равно sin 4x через функции от 2x?
✅ Верно! Это формула синуса двойного угла, где аргумент равен 2x.
❌ Это формула косинуса. Для синуса: sin 2α = 2 sin α cos α. Здесь α = 2x.
Вопрос 2. Как правильно записать cos² α без квадрата?
✅ Верно! Формула понижения степени для косинуса: плюс перед косинусом.
❌ Минус используется для синуса в квадрате. Для косинуса: (1 + cos 2α)/2.
Вопрос 3. Если 3x = π/3 + 2πk, чему равно x?
✅ Верно! Делить на коэффициент нужно весь ряд: π/3 : 3 и 2πk : 3.
❌ Нужно разделить на 3 каждое слагаемое: π/3 ÷ 3 = π/9, 2πk ÷ 3 = 2πk/3.
Алгоритм решения:
1. Приведите всё к одному виду (формулы приведения, если нужно).
2. Сведите аргументы к одному углу (формулы двойного угла).
3. Сгруппируйте слагаемые или примените формулы понижения степени.
4. Разложите на множители → произведение = 0.
5. Решите простейшие уравнения, аккуратно деля на коэффициент при x.
Ключевые формулы
sin 2α = 2sinα cosα
cos 2α = 2cos²α − 1
cos²α = (1 + cos 2α)/2

Формулы двойного угла

sin 2α = 2sinα cosα, cos 2α = 2cos²α − 1 = 1 − 2sin²α. Позволяют свести разные аргументы к одному.

Понижение степени

sin²α = (1 − cos 2α)/2, cos²α = (1 + cos 2α)/2. Убирают квадраты без извлечения корней.

Метод группировки

Разбиваем 4 слагаемых на пары, выносим общие множители, затем выносим общую скобку. Работает в 90% задач ЕГЭ №13.

Отбор корней

Всегда делите весь ряд (включая период) на коэффициент при x. Рисуйте окружность для визуального контроля.
