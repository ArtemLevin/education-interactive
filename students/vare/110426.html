ЗАМЕНА. ОДЗ. МЕТОД ИНТЕРВАЛОВ.

Тема занятия: логарифмические и показательные неравенства, в которых нужно заметить повторяющийся фрагмент, аккуратно перейти к новой переменной, решить неравенство по этой переменной и грамотно сделать обратную замену.

Встроенные настройки формата

Главная мысль Не домножайте неравенство на переменное выражение, пока не знаете его знак. Но после замены t = aˣ уже можно, потому что t > 0.

Разбираем типичное логарифмическое неравенство, где одна и та же часть lg x повторяется несколько раз.

1/(5 − lg x) + 2/(1 + lg x) − 1 < 0

Это удобный шаблон: сначала Вы выписываете ограничения, затем делаете замену t = lg x, сводите всё к одной дроби и решаете рациональное неравенство на оси t.

Что нужно найти: множество всех x, при которых исходное неравенство истинно.

x > 0;    lg x ≠ 5;    lg x ≠ −1

Логарифм существует только при x > 0. А ещё знаменатели не могут обращаться в ноль. Значит, на оси t потом обязательно появятся выколотые точки −1 и 5.

  • Неравенство нельзя решать, пока не зафиксированы запрещённые значения.
  • Именно они дают «дыры» на числовой прямой.
  • Это не косметика, а часть ответа.
t = lg x

После замены исходная задача становится проще по форме: исчезает логарифм, остаётся рациональное неравенство по переменной t.

1/(5 − t) + 2/(1 + t) − 1 < 0

Это главный методический трюк: Вы не «решаете логарифм», а временно переносите задачу на более знакомую территорию — к дробям и методу интервалов.

Идея проста: привести всё к одному знаменателю и получить вид P(t)/Q(t) < 0.

(1 + t + 2(5 − t) − (5 − t)(1 + t)) / ((5 − t)(1 + t)) < 0
  • Нельзя просто «умножить всё на знаменатель», потому что знак знаменателя зависит от t.
  • Зато можно привести к общему знаменателю: это безопасная операция.
  • Именно знаменатель потом даст выколотые точки на оси.
t² − 5t + 6 = (t − 2)(t − 3)
((t − 2)(t − 3)) / ((5 − t)(1 + t)) < 0

Критические точки на оси t: −1, 2, 3, 5.

После расстановки знаков получаем решение на языке t:

t ∈ (−∞, −1) ∪ (2, 3) ∪ (5, +∞)
lg x < −1   или   2 < lg x < 3   или   lg x > 5

Переход к x делается уже по стандартным правилам логарифма:

x < 10⁻¹,   10² < x < 10³,   x > 10⁵

Но ответ нужно ещё пересечь с условием x > 0. Поэтому первый кусок становится не «x < 0.1», а 0 < x < 0.1.

Проверяем три вещи: знак неравенства, выколотые точки, ограничение x > 0.

  • Точки −1 и 5 на оси t не включаются.
  • Точки 2 и 3 тоже не включаются, потому что неравенство строгое.
  • На оси x нельзя заходить в x ≤ 0.

Если пользователь случайно запишет x < 0.1 без ограничения x > 0, ответ станет шире нужного и будет неверным.

t = 3ˣ,   t > 0

Например, из занятия: после замены можно получить выражение вида

8 − 4/t < 3t

и тогда домножить на t уже безопасно, потому что t всегда положительно.

8t − 4 < 3t²   ⟺   3t² − 8t + 4 > 0

Это тонкий, но очень важный методический момент: в логарифмических дробях знак знаменателя неизвестен, а в замене t = aˣ знак известен заранее.

Печатный конспект

  • Сначала ОДЗ, затем замена, затем одна дробь, затем ось знаков, затем обратная замена.
  • Знаменатель в неравенстве не выбрасываем.
  • Если t = aˣ, то t > 0, поэтому домножение на t разрешено.

Типичные ошибки

  1. Забыли ОДЗ. Под логарифмом должен стоять положительный аргумент, а знаменатель не может быть равен нулю.
  2. Выкинули знаменатель после приведения к общей дроби. В неравенствах так нельзя.
  3. Домножили на выражение с переменной, не зная его знака.
  4. Сделали замену, но не вернулись к x. Ответ на языке t — это ещё не финал.
  5. Не пересекли ответ с x > 0. Из-за этого первый интервал часто записывают неверно.
  6. Неправильно сравнили логарифмы и забыли, что lg x < −1 означает x < 0.1, а не x < −1.

Наведение на шаги слева и на объекты справа синхронизировано: страница специально показывает, где именно рождается каждая типичная ошибка.

  • Что ищем: множество x, а не промежуточную переменную t.

Без вырождений С ответом С короткой идеей

Идея решения

Повторяющийся фрагмент → новая переменная → стандартное неравенство → метод интервалов → обратная замена.

Для логарифмической дроби главным становится контроль ОДЗ. Для показательной замены главным становится факт t > 0.

Наведите на или на , чтобы быстро перейти к нужному фокусу.

Памятка для печати

  • Всегда сначала отмечайте, где выражение не имеет смысла.
  • Замену делайте только там, где повторяющийся элемент действительно один и тот же.
  • После решения по t обязательно вернитесь к x.
  • Если t = aˣ, то сразу фиксируйте t > 0.
