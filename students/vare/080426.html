Схема решения

7 этапов: от условий к ответу

Дано

Что вообще перед нами?

Логарифмическое неравенство — это не только алгебра, но и вопрос допустимости: есть ли право брать логарифм, можно ли сравнивать два логарифма напрямую, удобно ли ввести замену t = logaf(x).
Сначала определяем тип: простейшее, с разными основаниями, вложенное или квадратное по логарифму.
Определите тип неравенства.
Смысл условий

Что требует ОДЗ?

Для каждого логарифма одновременно нужны три вещи: аргумент > 0, основание > 0, основание ≠ 1. Для вложенных логарифмов это делается последовательно: внутренний логарифм должен существовать, а его значение — ещё и подходить для внешнего логарифма.
Аргумент каждого логарифма положителен; основание положительно и не равно 1.
Сначала ОДЗ.
Интерпретация

Что даёт основание?

Если a > 1, функция y = logax возрастает, поэтому знак после снятия логарифмов сохраняется. Если 0 < a < 1, функция убывает, поэтому знак меняется на противоположный. Это главный переключатель всей темы.
a > 1 — знак сохраняется; 0 < a < 1 — знак меняется.
Следите за направлением знака.
Идея

Ключевая идея решения

Нужно свести задачу к знакомому виду: сравнить логарифмы с одинаковым основанием, вынести множитель через свойства логарифмов, сделать замену t, либо снять внешний логарифм и двигаться внутрь. После этого остаётся обычное алгебраическое неравенство или система.

Решение

Алгебраический этап

После снятия логарифмов решаем линейное, квадратное, рациональное или составное неравенство. При замене t решаем неравенство относительно t, затем возвращаемся к x через систему или совокупность. На этом этапе часто теряют знак или неверно оформляют обратный переход.

Что нужно проверить обязательно?

Нужно пересечь найденное решение с ОДЗ. Если в ответе есть корни и иррациональные границы, их порядок нужно доказать, а не угадывать. Если получено пустое множество, это тоже полноценный ответ.

Что дано и что искать

Школьная карта темы

Что дано

логарифмы с одинаковым основанием основание меньше 1 разные основания вложенные логарифмы квадратное по log

Что найти

Множество всех x, для которых выражение имеет смысл и исходное неравенство истинно.

Формула мысли: ответ = решение после преобразований ∩ ОДЗ.
База

Ключевые правила и свойства

ОДЗ

  • Аргумент логарифма строго положителен.
  • Основание строго положительно.
  • Основание не равно 1.
Например, для logx(2x − 1) нужны условия x > 0, x ≠ 1, 2x − 1 > 0.

Свойства

  • loga(bc) = logab + logac
  • loga(b/c) = logab − logac
  • loga(bp) = p·logab
  • logab = logcb / logca

a > 1

Функция возрастает, знак не меняется.

logaf(x) > logag(x) ⟺ f(x) > g(x)

0 < a < 1

Функция убывает, знак меняется.

logaf(x) > logag(x) ⟺ f(x) < g(x)
Идея решения

Как выбирать метод

Простейшее неравенство

logaf(x) ≷ c → представьте число c как логарифм с тем же основанием и сравните аргументы.

Разные основания

Переходите к одному основанию только если это реально упрощает задачу. Иногда проще ввести натуральный логарифм или оставить переход в виде дроби.

Вложенные логарифмы

Сначала условия существования внутреннего логарифма, потом снимаем внешний, потом внутренний.

Замена переменной

Если выражение повторяется, положите t = logax и решайте обычное квадратное или рациональное неравенство по t.

Пошаговый разбор

Три типовых примера

Пример 1. log2(x − 3) < 3

Разбор

  1. ОДЗ: x − 3 > 0 ⇒ x > 3.
  2. Представим 3 = log28.
  3. Получаем log2(x − 3) < log28.
  4. Так как 2 > 1, знак сохраняется: x − 3 < 8.
  5. x < 11, пересечение с ОДЗ даёт x ∈ (3; 11).

Проверка

Возьмём x = 5. Тогда log22 = 1 < 3. Подходит.

Возьмём x = 12. Тогда log29 > 3. Не подходит.

Ответ: (3; 11).
Пример 2. log0,22x − log0,2x − 6 < 0

Разбор

  1. ОДЗ: x > 0.
  2. Положим t = log0,2x.
  3. Тогда t² − t − 6 < 0.
  4. (t − 3)(t + 2) < 0 ⇒ t ∈ (−2; 3).
  5. Возвращаемся: −2 < log0,2x < 3.
  6. Так как основание меньше 1, после потенцирования знак меняется: 25 > x > 0,008.

Вывод

Ответ: x ∈ (0,008; 25).

Когда решение для t — интервал, обратно к x возвращаемся через систему двух неравенств.
Пример 3. log1/3(log2(x²)) ≤ 1

Разбор

  1. Для внешнего логарифма нужно log2(x²) > 0, значит x² > 1.
  2. Основание 1/3 < 1, знак меняется: log2(x²) ≥ 1/3.
  3. Так как 2 > 1, получаем x² ≥ 21/3.
  4. Число 21/3 > 1, значит условие ОДЗ уже выполнено автоматически.

Ответ

x ∈ (−∞; −21/6] ∪ [21/6; +∞), потому что x² ≥ 21/3.

Если после снятия логарифма появляется условие сильнее ОДЗ, отдельно его уже не решают второй раз.
Типичные ошибки

Где чаще всего теряют баллы

  • забывают ОДЗ или проверяют его не до конца;
  • не меняют знак при основании между 0 и 1;
  • неправильно оформляют обратную замену;
  • сокращают выражения без учёта знака множителя;
  • не замечают, что ответ может быть пустым множеством.

Очень важный антипример

В задаче log5x − log5(x − 2) < log0,23 получается

log5(x/(x − 2)) < log5(1/3).

При x > 2 выражение x/(x − 2) > 1, а 1/3 < 1. Значит неравенство невозможно.

Ответ здесь — ∅, а не интервал.

Памятка

Что помнить на экзамене

  1. ОДЗ выписываем сразу.
  2. Основание проверяем отдельно от аргумента.
  3. Увидели основание меньше 1 — мысленно включили «перевернуть знак».
  4. После замены по t обязательно возвращаемся к x корректно.
  5. Пустое множество — нормальный ответ.
