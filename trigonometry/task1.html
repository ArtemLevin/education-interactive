Задача 1: Базовые преобразования

Формулы приведения и выделение общего множителя

📌 Условие задачи

Решить тригонометрическое уравнение:

2 sin²(π/2 − x) + sin 2x = 0

💡 Идея решения

  1. Использовать формулу приведения для упрощения sin(π/2 − x)
  2. Применить формулу двойного угла для sin 2x
  3. Вынести общий множитель за скобку
  4. Решить два простых уравнения: cos x = 0 и tg x = −1
  5. Записать объединение серий корней

📘 Подробное решение

Шаг 1: Применение формулы приведения
1 Что дано: Уравнение содержит выражение sin(π/2 − x).

Геометрическая интерпретация: Угол (π/2 − x) является дополнительным до π/2. По формулам приведения синус дополнительного угла равен косинусу исходного угла.

Преобразование:
sin(π/2 − x) = cos x
Подставляем в уравнение:
2 cos² x + sin 2x = 0
Шаг 2: Формула двойного угла
2 Ключевая идея: Выразить sin 2x через sin x и cos x, чтобы получить однородное уравнение.

Формула двойного угла:
sin 2x = 2 sin x cos x
Получаем:
2 cos² x + 2 sin x cos x = 0
Шаг 3: Выделение общего множителя
3 Алгебраическое преобразование: Замечаем общий множитель 2 cos x в обоих слагаемых.

Выносим за скобку:
2 cos x (cos x + sin x) = 0
Произведение равно нулю, когда:
cos x = 0   или   cos x + sin x = 0
Шаг 4: Решение первого уравнения
4 Уравнение: cos x = 0

Решение: Косинус равен нулю в точках, где угол равен нечётному числу π/2.
x = π/2 + πk,    k ∈ ℤ
Шаг 5: Решение второго уравнения
5 Уравнение: cos x + sin x = 0

Преобразование: Переносим cos x вправо и делим на cos x (cos x ≠ 0, так как это случай из первого уравнения):
sin x = −cos x
tg x = −1
Решение:
x = −π/4 + πk,    k ∈ ℤ
Шаг 6: Проверка и ответ
6 Проверка: Обе серии корней удовлетворяют исходному уравнению. Подстановка подтверждает корректность.

Вывод: Объединяем две серии корней.

✅ Ответ

x = π/2 + πk   или   x = −π/4 + πk,    k ∈ ℤ

⚠️ Типичные ошибки

  • Неправильная формула приведения: sin(π/2 − x) = sin x (ошибка! должно быть cos x)
  • Потеря общего множителя: попытка решать квадратное уравнение вместо разложения на множители
  • Неполный ответ: забывают записать одну из двух серий корней
  • Ошибка в знаке: tg x = −1 даёт x = −π/4 + πk, а не π/4 + πk
  • Отсутствие параметра k: забывают указать, что k ∈ ℤ
