Задача 13: Уравнение cos 4x + cos 2x = 0

Формула суммы косинусов и объединение серий корней

📌 Условие задачи

Решить тригонометрическое уравнение:

cos 4x + cos 2x = 0

💡 Идея решения

  1. Применить формулу суммы косинусов: cos α + cos β = 2 cos((α+β)/2) · cos((α−β)/2)
  2. Преобразовать сумму в произведение
  3. Приравнять каждый множитель к нулю
  4. Решить два простых уравнения: cos 3x = 0 и cos x = 0
  5. Объединить серии корней, заметив, что одна содержится в другой

📘 Подробное решение

Шаг 1: Формула суммы косинусов
1 Что дано: Сумма двух косинусов с разными аргументами.

Ключевая идея: Использовать формулу суммы косинусов для преобразования суммы в произведение.

Формула:
cos α + cos β = 2 cos((α+β)/2) · cos((α−β)/2)
Шаг 2: Применение формулы
2 Обозначения: α = 4x, β = 2x

Вычисляем:
(α+β)/2 = (4x+2x)/2 = 3x
(α−β)/2 = (4x−2x)/2 = x
Применяем формулу:
cos 4x + cos 2x = 2 cos 3x · cos x
Шаг 3: Произведение равно нулю
3 Уравнение принимает вид:
2 cos 3x · cos x = 0
Произведение равно нулю, когда:
cos 3x = 0   или   cos x = 0
Шаг 4: Решение первого уравнения
4 Уравнение: cos 3x = 0

Решение: Косинус равен нулю в нечётных кратных π/2:
3x = π/2 + πk
Делим на 3:
x = π/6 + πk/3,    k ∈ ℤ
Шаг 5: Решение второго уравнения
5 Уравнение: cos x = 0

Решение:
x = π/2 + πk,    k ∈ ℤ
Шаг 6: Объединение серий
6 Анализ: Заметим, что серия x = π/2 + πk уже содержится в общей серии x = π/6 + πk/3.

Проверка: При k = 1 получаем x = π/6 + π/3 = π/2. При k = 4 получаем x = π/6 + 4π/3 = 3π/2 и т.д.

Вывод: Достаточно записать только общую серию.
Шаг 7: Проверка и ответ
7 Проверка: Подстановка подтверждает, что все значения из серии удовлетворяют исходному уравнению.

Вывод: Записываем окончательный ответ.

✅ Ответ

x = π/6 + πk/3,    k ∈ ℤ

⚠️ Типичные ошибки

  • Неправильная формула суммы: путают формулу суммы косинусов с другими формулами
  • Ошибка в знаках: забывают, что в формуле (α−β)/2, а не (β−α)/2
  • Неполное объединение: записывают обе серии корней отдельно, не замечая, что одна содержится в другой
  • Ошибка при делении: забывают разделить πk на 3 при переходе от 3x к x
  • Потеря параметра: забывают указать k ∈ ℤ
