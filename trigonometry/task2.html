Задача 2: Квадратное уравнение относительно sin x

Формулы приведения и решение квадратного уравнения

📌 Условие задачи

Решить тригонометрическое уравнение:

cos 2x − √2 · cos(3π/2 + x) − 1 = 0

💡 Идея решения

  1. Использовать формулу приведения для cos(3π/2 + x) = sin x
  2. Выразить cos 2x через sin x по формуле cos 2x = 1 − 2sin² x
  3. Получить квадратное уравнение относительно sin x
  4. Разложить на множители и решить два простых уравнения
  5. Записать три серии корней с учётом периодичности

📘 Подробное решение

Шаг 1: Формула приведения для cos(3π/2 + x)
1 Что дано: Уравнение содержит выражение cos(3π/2 + x).

Геометрическая интерпретация: Угол (3π/2 + x) находится в четвёртой четверти при малых положительных x. Косинус в четвёртой четверти положителен.

Формула приведения: При добавлении 3π/2 косинус превращается в синус с сохранением знака:
cos(3π/2 + x) = sin x
Подставляем в уравнение:
cos 2x − √2 · sin x − 1 = 0
Шаг 2: Выражаем cos 2x через sin x
2 Ключевая идея: Привести уравнение к виду, содержащему только sin x, чтобы получить квадратное уравнение.

Формула двойного угла:
cos 2x = 1 − 2sin² x
Получаем:
(1 − 2sin² x) − √2 · sin x − 1 = 0
Упрощаем:
−2sin² x − √2 · sin x = 0
Шаг 3: Разложение на множители
3 Алгебраическое преобразование: Выносим общий множитель sin x за скобку.

Выносим за скобку:
sin x · (2sin x + √2) = 0
Произведение равно нулю, когда:
sin x = 0   или   2sin x + √2 = 0
Второе уравнение упрощаем:
sin x = −√2 / 2
Шаг 4: Решение первого уравнения sin x = 0
4 Уравнение: sin x = 0

Решение: Синус равен нулю в точках, кратных π.
x = πk,    k ∈ ℤ
Шаг 5: Решение второго уравнения sin x = −√2/2
5 Уравнение: sin x = −√2 / 2

Геометрическая интерпретация: Значение −√2/2 соответствует углу −π/4 (четвёртая четверть) и углу π + π/4 = 5π/4 (третья четверть).

Общее решение:
x = −π/4 + 2πk,    k ∈ ℤ
x = 5π/4 + 2πk,    k ∈ ℤ
Шаг 6: Проверка и ответ
6 Проверка: Все три серии корней удовлетворяют исходному уравнению.

Проверка для x = πk:
cos(2πk) − √2·cos(3π/2 + πk) − 1 = 1 − √2·0 − 1 = 0 ✓

Проверка для x = −π/4 + 2πk:
cos(−π/2) − √2·cos(5π/4) − 1 = 0 − √2·(−√2/2) − 1 = 1 − 1 = 0 ✓

Вывод: Объединяем три серии корней.

✅ Ответ

x = πk,   x = −π/4 + 2πk,   x = 5π/4 + 2πk,   k ∈ ℤ

⚠️ Типичные ошибки

  • Неправильная формула приведения: cos(3π/2 + x) = −sin x (ошибка! должно быть +sin x)
  • Неверная формула cos 2x: используют cos 2x = 2cos² x − 1 вместо 1 − 2sin² x
  • Потеря одного корня: забывают одну из двух серий для sin x = −√2/2
  • Ошибка в знаке: пишут π/4 вместо −π/4 для отрицательного значения синуса
  • Неправильный период: пишут πk вместо 2πk для уравнения sin x = a
