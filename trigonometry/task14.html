Задача 14: Метод группировки

Выделение общего множителя и решение совокупности уравнений

📌 Условие задачи

Решить тригонометрическое уравнение:

2sin 2x − 4cos x + 3sin x − 3 = 0

💡 Идея решения

  1. Применить формулу двойного угла для sin 2x
  2. Сгруппировать слагаемые попарно
  3. Вынести общие множители в каждой группе
  4. Выделить общий множитель (sin x − 1)
  5. Решить два простых уравнения: sin x = 1 и cos x = −3/4

📘 Подробное решение

Шаг 1: Формула двойного угла
1 Что дано: Уравнение содержит sin 2x, cos x и sin x.

Ключевая идея: Выразить sin 2x через sin x и cos x, чтобы получить однородное выражение.

Формула:
sin 2x = 2sin x cos x
Подставляем:
2·(2sin x cos x) − 4cos x + 3sin x − 3 = 0
4sin x cos x − 4cos x + 3sin x − 3 = 0
Шаг 2: Группировка слагаемых
2 Алгебраическое преобразование: Разбиваем четыре слагаемых на две группы:
(4sin x cos x − 4cos x) + (3sin x − 3) = 0
В первой группе общий множитель 4cos x, во второй — 3.
Шаг 3: Вынесение общих множителей
3 Первая группа:
4sin x cos x − 4cos x = 4cos x(sin x − 1)
Вторая группа:
3sin x − 3 = 3(sin x − 1)
Получаем:
4cos x(sin x − 1) + 3(sin x − 1) = 0
Шаг 4: Выделение общего множителя
4 Ключевой момент: Замечаем общий множитель (sin x − 1) в обоих слагаемых.

Выносим за скобку:
(sin x − 1)(4cos x + 3) = 0
Произведение равно нулю, когда:
sin x − 1 = 0   или   4cos x + 3 = 0
Шаг 5: Решение первого уравнения
5 Уравнение: sin x − 1 = 0, то есть sin x = 1

Решение: Синус равен 1 в точке π/2 плюс период:
x = π/2 + 2πk,    k ∈ ℤ
Шаг 6: Решение второго уравнения
6 Уравнение: 4cos x + 3 = 0

Преобразование:
cos x = −3/4
Решение: Косинус отрицателен во II и III четвертях:
x = arccos(−3/4) + 2πk
или x = 2π − arccos(−3/4) + 2πk,    k ∈ ℤ
Шаг 7: Проверка и ответ
7 Проверка: Все три серии корней удовлетворяют исходному уравнению. Значение |−3/4| ≤ 1, поэтому arccos существует.

Вывод: Объединяем все серии корней.

✅ Ответ

x = π/2 + 2πk,
x = arccos(−3/4) + 2πk,
x = 2π − arccos(−3/4) + 2πk,    k ∈ ℤ

⚠️ Типичные ошибки

  • Неправильная формула для sin 2x: забывают коэффициент 2 в формуле sin 2x = 2sin x cos x
  • Ошибка при группировке: неправильно группируют слагаемые, не замечая общей структуры
  • Потеря одного из случаев: решают только одно уравнение из совокупности
  • Ошибка в знаке: cos x = −3/4, а не 3/4
  • Неполная запись ответа: забывают вторую серию для arccos или не указывают k ∈ ℤ
