Задача 6: Логарифмическое тригонометрическое уравнение

Решение уравнения вида log₁/₃(2sin²x − 3cos2x + 6) = −2

📋 Оглавление ← Задача 5 Задача 7 →
log¹⁄₃(2sin²x − 3cos2x + 6) = −2
💡 Идея решения
  1. Используем определение логарифма: logab = c ⇔ b = ac
  2. Переходим от логарифмического уравнения к тригонометрическому
  3. Выражаем cos2x через sin²x по формуле cos2x = 1 − 2sin²x
  4. Получаем простое уравнение относительно sin²x
  5. Находим sin x и записываем общий ответ
1Что дано

Логарифмическое уравнение с тригонометрическими функциями:

log¹⁄₃(2sin²x − 3cos2x + 6) = −2

Основание логарифма: a = ¹⁄₃ (0 < a < 1)

Подлогарифмическое выражение: f(x) = 2sin²x − 3cos2x + 6

2Область допустимых значений (ОДЗ)
⚠️ Условие существования логарифма:

Подлогарифмическое выражение должно быть строго положительным:

2sin²x − 3cos2x + 6 > 0

Проверим это условие после нахождения потенциальных корней.

3Применяем определение логарифма

По определению логарифма:

logaf(x) = b  ⟺  f(x) = ab

В нашем случае:

2sin²x − 3cos2x + 6 = (¹⁄₃)−2

Вычисляем правую часть:

(¹⁄₃)−2 = 3² = 9

Получаем тригонометрическое уравнение:

2sin²x − 3cos2x + 6 = 9
4Упрощаем уравнение

Переносим 9 в левую часть:

2sin²x − 3cos2x + 6 − 9 = 0
2sin²x − 3cos2x − 3 = 0
5Используем формулу двойного угла

Ключевая идея: выразить cos2x через sin²x, чтобы получить уравнение с одной тригонометрической функцией.

Формула: cos2x = 1 − 2sin²x

Подставляем в уравнение:

2sin²x − 3(1 − 2sin²x) − 3 = 0
6Раскрываем скобки и приводим подобные

Раскрываем скобки:

2sin²x − 3 + 6sin²x − 3 = 0

Приводим подобные слагаемые:

8sin²x − 6 = 0
7Решаем уравнение относительно sin²x

Переносим −6 в правую часть:

8sin²x = 6

Делим на 8:

sin²x = ⁶⁄₈ = ³⁄₄
8Находим sin x

Извлекаем квадратный корень:

sin x = ±√(³⁄₄) = ±√3⁄2

Получаем два случая:

sin x = √3⁄2   или   sin x = −√3⁄2
9Решаем простейшие тригонометрические уравнения

Случай 1: sin x = √3⁄2

x = π⁄3 + 2πk   или   x = 2π⁄3 + 2πk,   k ∈ ℤ

Случай 2: sin x = −√3⁄2

x = −π⁄3 + 2πk   или   x = −2π⁄3 + 2πk,   k ∈ ℤ

Можно объединить решения в две серии:

x = π⁄3 + πk   или   x = 2π⁄3 + πk,   k ∈ ℤ
10Проверка ОДЗ

Проверим, что найденные значения удовлетворяют условию 2sin²x − 3cos2x + 6 > 0.

Для всех найденных решений sin²x = ³⁄₄, тогда:

cos2x = 1 − 2sin²x = 1 − 2·³⁄₄ = 1 − ³⁄₂ = −¹⁄₂

Подставляем в подлогарифмическое выражение:

2·³⁄₄ − 3·(−¹⁄₂) + 6 = ³⁄₂ + ³⁄₂ + 6 = 3 + 6 = 9 > 0

✅ Условие ОДЗ выполнено для всех найденных значений.

✓ Ответ
x = π⁄3 + πk   или   x = 2π⁄3 + πk,   k ∈ ℤ
⚠️ Типичные ошибки
Ошибка 1: Забыли проверить ОДЗ. Всегда проверяйте, что подлогарифмическое выражение положительно!
Ошибка 2: Неправильное вычисление (¹⁄₃)−2. Помните: a−n = 1⁄an, поэтому (¹⁄₃)−2 = 3² = 9.
Ошибка 3: Неверная формула для cos2x. Правильно: cos2x = 1 − 2sin²x (а не 2sin²x − 1).
Ошибка 4: Потеря одного из случаев при извлечении корня. Из sin²x = ³⁄₄ следует sin x = ±√3⁄2.
Ошибка 5: Неправильная запись общего ответа. Для sin x = ±√3⁄2 получаем две серии с периодом π.
