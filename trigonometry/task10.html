Задача 10: Комбинированное уравнение с модулем

Решение тригонометрического уравнения с модулем и отбором корней на промежутке

📋 Условие задачи

а) Решите уравнение:

|cosx| = cosx −2sinx

б) Найдите все корни этого уравнения, принадлежащие промежутку −3π2;5π2.

💡 Идея решения

  1. Раскрыть модуль по определению — рассмотреть два случая: cos x ≥ 0 и cos x < 0
  2. В каждом случае решить полученное тригонометрическое уравнение
  3. Проверить полученные корни на соответствие условию раскрытия модуля
  4. Для пункта б) отобрать корни, принадлежащие заданному промежутку

1Что дано

Уравнение содержит модуль косинуса. По определению модуля:

⚠️ Определение модуля

|cosx| = cosx,еслиcosx≥0 −cosx,еслиcosx<0

2Случай 1: cos x ≥ 0

Когда cosx≥0, модуль раскрывается без изменения знака:

cosx=cosx−2sinx

Упрощаем уравнение:

0=−2sinx

Отсюда:

sinx=0

Решение этого уравнения:

x=πk,k∈Z

Проверка условия cos x ≥ 0:

  • При чётных k = 2n: x=2πn, тогда cos2πn=1≥0 ✓
  • При нечётных k = 2n+1: x=π+2πn, тогда cosπ+2πn=−1<0 ✗

Вывод: В первом случае подходят только чётные значения k:

x=2πn,n∈Z

3Случай 2: cos x < 0

Когда cosx<0, модуль раскрывается с изменением знака:

−cosx=cosx−2sinx

Переносим все члены в одну сторону:

2sinx=2cosx

Делим на 2:

sinx=cosx

Делим на cos x ≠ 0 (так как cos x < 0):

tgx=1

Решение этого уравнения:

x=π4+πk,k∈Z

Проверка условия cos x < 0:

  • При чётных k = 2n: x=π4+2πn, тогда cosπ4=22>0 ✗
  • При нечётных k = 2n+1: x=π4+π+2πn=5π4+2πn, тогда cos5π4=−22<0 ✓

Вывод: Во втором случае подходят только нечётные значения k:

x=5π4+2πn,n∈Z

4Объединение серий

Объединяем найденные серии корней из обоих случаев:

x=2πn,n∈Z
x=5π4+2πn,n∈Z

5Отбор корней на промежутке (пункт б)

Нужно найти корни, принадлежащие промежутку −3π2;5π2.

Числовые значения границ:

−3π2≈−4.71;5π2≈7.85

Первая серия: x = 2πn

  • n = −1: x=−2π≈−6.28 — не входит (−6.28<−4.71)
  • n = 0: x=0 — входит ✓
  • n = 1: x=2π≈6.28 — входит ✓
  • n = 2: x=4π≈12.57 — не входит

Вторая серия: x = 5π/4 + 2πn

  • n = −1: x=5π4−2π=−3π4≈−2.36 — входит ✓
  • n = 0: x=5π4≈3.93 — входит ✓
  • n = 1: x=13π4≈10.21 — не входит

✅ Ответ

а)

x=2πn,n∈Z;x=5π4+2πn,n∈Z

б)

0;π;−3π4;5π4

❌ Типичные ошибки

  • Неправильное раскрытие модуля — забывают рассмотреть оба случая или путают знаки
  • Отсутствие проверки условия — после решения в каждом случае нужно проверить, удовлетворяют ли корни условию раскрытия модуля
  • Потеря одной из серий — часто забывают про вторую серию корней из случая cos x < 0
  • Ошибка при отборе корней — неверно определяют границы промежутка или пропускают корни на границах
  • Неполный ответ в пункте б) — забывают указать все четыре корня, принадлежащих промежутку
