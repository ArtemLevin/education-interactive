Задача 8: Показательное тригонометрическое уравнение

Решение уравнения вида a^(sin² x) = b^(sin 2x) · c

💡 Идея решения

  1. Привести обе части уравнения к одному основанию степени (в данном случае к основанию 5)
  2. Приравнять показатели степеней, так как основания равны
  3. Использовать формулу sin 2x = 2 sin x cos x для перехода к одинаковым аргументам
  4. Заменить единицу на sin² x + cos² x для получения однородного уравнения
  5. Разложить полученное выражение на множители и решить два простых уравнения

Дано уравнение:

125^(sin² x) = (√5)^(5 sin 2x) · 0,2

Пошаговое решение:

1 Приведём обе части к основанию 5:
125^(sin² x) = (5³)^(sin² x) = 5^(3 sin² x)

(√5)^(5 sin 2x) · 0,2 = (5^(1/2))^(5 sin 2x) · 5^(-1) = 5^((5/2) sin 2x) · 5^(-1) = 5^((5/2) sin 2x - 1)
2 Приравниваем показатели степеней:
3 sin² x = (5/2) sin 2x - 1
3 Подставляем формулу двойного угла:
sin 2x = 2 sin x cos x

Получаем:

3 sin² x = (5/2) · 2 sin x cos x - 1
3 sin² x = 5 sin x cos x - 1
4 Переносим все члены в одну сторону:
3 sin² x - 5 sin x cos x + 1 = 0
5 Заменяем единицу на основное тригонометрическое тождество:
1 = sin² x + cos² x

Тогда уравнение принимает вид:

3 sin² x - 5 sin x cos x + sin² x + cos² x = 0
4 sin² x - 5 sin x cos x + cos² x = 0
6 Раскладываем на множители:
(4 sin x - cos x)(sin x - cos x) = 0

Проверка раскрытия скобок:

4 sin² x - 4 sin x cos x - sin x cos x + cos² x = 4 sin² x - 5 sin x cos x + cos² x ✓
7 Решаем два полученных уравнения:
1) 4 sin x - cos x = 0 ⟹ 4 sin x = cos x ⟹ tg x = 1/4
x = arctg(1/4) + πk

2) sin x - cos x = 0 ⟹ sin x = cos x ⟹ tg x = 1
x = π/4 + πk

✓ Ответ:

x = arctg(1/4) + πk   или   x = π/4 + πk,   k ∈ Z

⚠️ Типичные ошибки

  • Неправильное приведение к основанию: забывают, что 125 = 5³, √5 = 5^(1/2), 0,2 = 5^(-1)
  • Ошибка при приравнивании показателей: пытаются приравнять сами степени, а не показатели
  • Неполное использование формул: забывают формулу sin 2x = 2 sin x cos x
  • Ошибка при разложении на множители: неправильно раскладывают квадратное выражение
  • Потеря корней: забывают записать оба случая из произведения или забывают +πk

📐 Ключевые формулы

a^m · a^n = a^(m+n)
(a^m)^n = a^(mn)
a^m = a^n ⟹ m = n (при a > 0, a ≠ 1)
sin 2x = 2 sin x cos x
sin² x + cos² x = 1
125 = 5³, √5 = 5^(1/2), 0,2 = 5^(-1)
