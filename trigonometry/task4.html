Задача 4: Универсальная тригонометрическая подстановка

Главная → Задания → Задача 4

Условие задачи

Решить тригонометрическое уравнение:

cos(2x) − sin(2x) = cos(x) + sin(x) + 1

💡 Идея решения

  • Применить универсальную тригонометрическую подстановку через тангенс половинного угла: t = tg(x/2)
  • Выразить все тригонометрические функции через t: sin(x), cos(x), sin(2x), cos(2x)
  • После подстановки получить рациональное уравнение относительно t
  • Разложить числитель на квадратные множители и решить два квадратных уравнения
  • Вернуться к переменной x через арктангенс и табличные значения
1Что дано

Нам дано уравнение, содержащее:

  • В левой части: косинус двойного угла cos(2x) и синус двойного угла sin(2x)
  • В правой части: косинус cos(x), синус sin(x) и константу 1

Уравнение содержит функции разных аргументов (x и 2x). Для решения таких уравнений удобно использовать универсальную подстановку через половинный угол.

2Ключевая идея: универсальная подстановка

Введём новую переменную:

t = tg(x/2)

Тогда все тригонометрические функции выражаются через t по формулам:

sin(x) = 2t/(1 + t²),     cos(x) = (1 − t²)/(1 + t²)

Для двойного угла получаем:

sin(2x) = 4t(1 − t²)/(1 + t²)²,     cos(2x) = (1 − 6t² + t⁴)/(1 + t²)²
3Подстановка в уравнение

Подставляем все выражения в исходное уравнение:

(1 − 6t² + t⁴)/(1 + t²)² − 4t(1 − t²)/(1 + t²)² = (1 − t²)/(1 + t²) + 2t/(1 + t²) + 1

Приводим правую часть к общему знаменателю (1 + t²):

(1 − t² + 2t + 1 + t²)/(1 + t²) = (2 + 2t)/(1 + t²)

Теперь умножаем обе части на (1 + t²)²:

(1 − 6t² + t⁴) − 4t(1 − t²) = (2 + 2t)(1 + t²)
4Преобразование к алгебраическому уравнению

Раскрываем скобки и приводим подобные:

1 − 6t² + t⁴ − 4t + 4t³ = 2 + 2t + 2t² + 2t³

Переносим всё в левую часть:

t⁴ + 2t³ − 8t² − 6t − 1 = 0

Это уравнение четвёртой степени можно разложить на два квадратных множителя:

(t² − 2t − 1)(t² + 4t + 1) = 0
5Решение квадратных уравнений

Получаем два случая:

Случай 1: t² − 2t − 1 = 0

D = 4 + 4 = 8,     t₁ = 1 + √2,     t₂ = 1 − √2

Случай 2: t² + 4t + 1 = 0

D = 16 − 4 = 12,     t₃ = −2 + √3,     t₄ = −2 − √3
6Возврат к переменной x

Вспоминаем, что t = tg(x/2). Поэтому:

Для t₁ = 1 + √2:

tg(x/2) = 1 + √2 = tg(3π/8)     ⇒     x/2 = 3π/8 + πk     ⇒     x = 3π/4 + 2πk

Для t₂ = 1 − √2:

tg(x/2) = 1 − √2 = tg(−π/8)     ⇒     x/2 = −π/8 + πk     ⇒     x = −π/4 + 2πk

Для t₃ = −2 + √3:

tg(x/2) = −2 + √3 = tg(−π/12)     ⇒     x/2 = −π/12 + πk     ⇒     x = −π/6 + 2πk

Для t₄ = −2 − √3:

tg(x/2) = −2 − √3 = tg(−5π/12)     ⇒     x/2 = −5π/12 + πk     ⇒     x = −5π/6 + 2πk
7Проверка ответа

Проверим подстановкой найденные значения. Все четыре серии решений получены из табличных значений тангенса, поэтому они корректны.

Также нужно убедиться, что знаменатель 1 + t² ≠ 0, что очевидно верно для всех действительных t.

Особые случаи (когда x/2 = π/2 + πn, т.е. когда тангенс не определён) нужно проверить отдельно. При x = π + 2πn:

cos(2π) − sin(2π) = 1 − 0 = 1
cos(π) + sin(π) + 1 = −1 + 0 + 1 = 0

1 ≠ 0, значит эти значения не являются решениями, и мы ничего не потеряли при использовании универсальной подстановки.

📌 Ответ

x = 3π/4 + 2πk,    x = −π/4 + 2πk,    x = −π/6 + 2πk,    x = −5π/6 + 2πk,    k ∈ ℤ

⚠️ Типичные ошибки

Ошибка 1: Неправильное применение формул универсальной подстановки для двойного угла. Нужно аккуратно выводить sin(2x) и cos(2x) через t.
Ошибка 2: Потеря решений при проверке особых случаев. Всегда проверяйте, не потеряли ли вы значения, где tg(x/2) не определён.
Ошибка 3: Ошибки при разложении многочлена четвёртой степени на множители. Рекомендуется проверять разложение умножением обратно.
Ошибка 4: Неправильный переход от tg(x/2) = a к x. Помните: x/2 = arctg(a) + πk, значит x = 2·arctg(a) + 2πk.

💡 Совет

Универсальная подстановка особенно эффективна, когда в уравнении присутствуют функции разных аргументов (x, 2x, 3x и т.д.). Она сводит тригонометрическое уравнение к алгебраическому.
