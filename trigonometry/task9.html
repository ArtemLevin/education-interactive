Задача 9: Уравнение с областью допустимых значений

Решение рационального тригонометрического уравнения с предварительным выписыванием ОДЗ

📋 Условие задачи

4sin2(7π2−x) − 11cosx + 6 = 0

💡 Идея решения

  1. Выписать область допустимых значений — знаменатели не должны обращаться в ноль
  2. Использовать формулу приведения для упрощения sin(7π2−x)
  3. Свести уравнение к квадратному относительно cosx
  4. Проверить корни на соответствие ОДЗ и условию |cosx|≤1

1Что дано

Рациональное тригонометрическое уравнение с двумя знаменателями:

  • Первый знаменатель: sin2(7π2−x)
  • Второй знаменатель: cosx

2Область допустимых значений

⚠️ ОДЗ — первый и самый важный шаг!

Знаменатели не должны обращаться в ноль:

sin2(7π2−x)≠0 cosx≠0

Это означает:

  • sin(7π2−x)≠0
  • cosx≠0

3Формула приведения

Упростим выражение в первом знаменателе. Используем формулу приведения:

sin(7π2−x) = sin(3π2−x) = −cosx

Следовательно:

sin2(7π2−x) = cos2x

4Преобразование уравнения

Подставляем полученное выражение в исходное уравнение:

4cos2x − 11cosx + 6 = 0

5Замена переменной

Обозначим t=cosx, где t≠0. Получаем:

4t2 − 11t + 6 = 0

6Решение квадратного уравнения

Умножаем на t2:

4−11t+6t2 = 0

Приводим к стандартному виду:

6t2−11t+4 = 0

Разлагаем на множители:

3t−4 2t−1 = 0

Отсюда:

t=43 или t=12

7Проверка корней

Возвращаемся к переменной x:

  • cosx=43 — невозможно, так как |cosx|≤1
  • cosx=12 — подходит, так как |12|<1

8Решение простейшего уравнения

Решаем уравнение cosx=12:

x=±arccos12+2πk

Так как arccos12=π3, получаем:

x=±π3+2πk,k∈Z

9Проверка по ОДЗ

Проверяем найденные значения на соответствие ОДЗ:

  • cosπ3=12≠0 ✓
  • sin(7π2−π3)=sin19π6=−12≠0 ✓

Все условия ОДЗ выполнены.

✅ Ответ

x=π3+2πk или x=−π3+2πk,k∈Z

❌ Типичные ошибки

  • Не выписали ОДЗ — могли включить посторонние корни, где знаменатель равен нулю
  • Неправильно применили формулу приведения — знак минус часто теряется при преобразовании sin(7π2−x)
  • Забыли проверить |cosx|≤1 — корень t=43 нужно отбросить
  • Потеряли один из знаков — ответ должен содержать оба значения ±π3
  • Не проверили окончательный ответ по ОДЗ — всегда нужно убедиться, что найденные значения не обращают знаменатели в ноль
