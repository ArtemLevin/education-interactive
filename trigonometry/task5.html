📘 Задача 5: Логарифмическое тригонометрическое уравнение

Главная → Задания → Задача 5

📌 Условие задачи

Решить уравнение:

log₁/₂(3cos 2x − 2cos² x + 5) = −2

💡 Идея решения

  • Использовать определение логарифма для перехода от логарифмического уравнения к алгебраическому
  • Применить формулу двойного угла для cos 2x
  • Свести уравнение к квадратному относительно cos² x
  • Проверить область допустимых значений подлогарифмического выражения
  • Записать ответ с учётом периодичности косинуса

🔍 Пошаговое решение

1Что дано

Логарифмическое уравнение с основанием ½ и тригонометрическим выражением под знаком логарифма.

Уравнение имеет вид logₐ f(x) = b, где:

  • a = ½ — основание логарифма (0 < a < 1)
  • f(x) = 3cos 2x − 2cos² x + 5 — подлогарифмическое выражение
  • b = −2 — значение логарифма

Перед нами классическое логарифмическое уравнение вида logₐ f(x) = b. Важно помнить:

  • Основание логарифма a = ½ должно быть положительным и не равным 1 — это условие выполнено
  • Подлогарифмическое выражение f(x) должно быть строго положительным: f(x) > 0
  • Поскольку основание меньше 1, логарифмическая функция убывает
Область допустимых значений:
3cos 2x − 2cos² x + 5 > 0
2Применяем определение логарифма

По определению логарифма:

3cos 2x − 2cos² x + 5 = (½)⁻² = 4

Определение логарифма гласит:

logₐ f(x) = b ⇔ f(x) = aᵇ

Применяем к нашему уравнению:

3cos 2x − 2cos² x + 5 = (½)⁻²

Вычисляем правую часть:

(½)⁻² = (2⁻¹)⁻² = 2² = 4

Ключевой шаг решения — переход от логарифмической формы к показательной. Формула перехода:

logₐ f(x) = b ⇔ f(x) = aᵇ, при условиях a > 0, a ≠ 1, f(x) > 0

В нашем случае:

log₁/₂(3cos 2x − 2cos² x + 5) = −2 ⇔ 3cos 2x − 2cos² x + 5 = (½)⁻²

Вычисление степени с отрицательным показателем:

(½)⁻² = 1 / (½)² = 1 / (¼) = 4
✓ Проверка ОДЗ будет выполнена после нахождения корней
3Упрощаем уравнение

Переносим 4 в левую часть:

3cos 2x − 2cos² x + 1 = 0

Из равенства 3cos 2x − 2cos² x + 5 = 4 вычитаем 4 из обеих частей:

3cos 2x − 2cos² x + 5 − 4 = 0
3cos 2x − 2cos² x + 1 = 0

После применения определения логарифма получили алгебраическое уравнение. Приводим его к стандартному виду:

3cos 2x − 2cos² x + 5 = 4

Вычитаем 4 из обеих частей уравнения:

3cos 2x − 2cos² x + 5 − 4 = 4 − 4
3cos 2x − 2cos² x + 1 = 0

Теперь перед нами тригонометрическое уравнение, содержащее cos 2x и cos² x.

4Применяем формулу двойного угла

Используем формулу cos 2x = 2cos² x − 1:

3(2cos² x − 1) − 2cos² x + 1 = 0

Формула двойного угла для косинуса:

cos 2x = 2cos² x − 1

Подставляем в уравнение:

3(2cos² x − 1) − 2cos² x + 1 = 0

Для сведения уравнения к одному тригонометрическому функци используем одну из трёх форм формулы двойного угла:

cos 2x = 2cos² x − 1 = 1 − 2sin² x = cos² x − sin² x

Поскольку в уравнении уже есть cos² x, выбираем форму:

cos 2x = 2cos² x − 1

Подставляем эту замену в уравнение:

3(2cos² x − 1) − 2cos² x + 1 = 0

📐 Формула двойного угла

cos 2x = 2cos² x − 1 — позволяет выразить косинус двойного угла через квадрат косинуса одинарного угла

5Раскрываем скобки и приводим подобные
6cos² x − 3 − 2cos² x + 1 = 0
4cos² x − 2 = 0

Раскрываем скобки:

3 · 2cos² x − 3 · 1 − 2cos² x + 1 = 0
6cos² x − 3 − 2cos² x + 1 = 0

Приводим подобные слагаемые:

(6 − 2)cos² x + (−3 + 1) = 0
4cos² x − 2 = 0

Выполняем алгебраические преобразования пошагово:

Шаг 5.1: Раскрываем скобки, умножая каждый член в скобках на 3:

3(2cos² x − 1) = 6cos² x − 3

Шаг 5.2: Записываем полное уравнение:

6cos² x − 3 − 2cos² x + 1 = 0

Шаг 5.3: Группируем подобные слагаемые:

  • Члены с cos² x: 6cos² x − 2cos² x = 4cos² x
  • Свободные члены: −3 + 1 = −2
4cos² x − 2 = 0
6Решаем полученное уравнение
4cos² x = 2
cos² x = ½

Переносим −2 в правую часть:

4cos² x = 2

Делим обе части на 4:

cos² x = ½

Решаем простое алгебраическое уравнение относительно cos² x:

4cos² x − 2 = 0

Прибавляем 2 к обеим частям:

4cos² x = 2

Делим обе части на коэффициент 4:

cos² x = 2/4 = ½

Теперь нужно найти все x, для которых квадрат косинуса равен ½.

7Находим значения x
cos x = ±√(½) = ±√2/2
x = π/4 + πk/2, k ∈ ℤ

Извлекаем квадратный корень:

cos x = ±√(½) = ±1/√2 = ±√2/2

Это соответствует углам, где косинус равен ±√2/2:

  • cos x = √2/2 ⇒ x = ±π/4 + 2πk
  • cos x = −√2/2 ⇒ x = ±3π/4 + 2πk

Объединяем серии:

x = π/4 + πk/2, k ∈ ℤ

Решаем уравнение cos² x = ½:

Шаг 7.1: Извлекаем квадратный корень из обеих частей:

|cos x| = √(½)

Шаг 7.2: Упрощаем правую часть:

√(½) = 1/√2 = √2/2

Шаг 7.3: Получаем два случая:

cos x = √2/2 или cos x = −√2/2

Шаг 7.4: Решаем каждое уравнение:

  • cos x = √2/2 ⇒ x = ±π/4 + 2πn, n ∈ ℤ
  • cos x = −√2/2 ⇒ x = ±3π/4 + 2πm, m ∈ ℤ

Шаг 7.5: Объединяем решения:

Все четыре серии можно записать компактно:

x = π/4 + πk/2, k ∈ ℤ

Проверка: при k = 0, 1, 2, 3 получаем π/4, 3π/4, 5π/4, 7π/4 — все нужные значения.

8Проверяем область допустимых значений
✓ Подлогарифмическое выражение равно 4 > 0 — условие выполнено

Проверяем, что подлогарифмическое выражение положительно:

По условию уравнения:

3cos 2x − 2cos² x + 5 = 4

4 > 0, значит все найденные значения x удовлетворяют ОДЗ.

Обязательный этап решения логарифмических уравнений — проверка области допустимых значений.

Условие ОДЗ:

3cos 2x − 2cos² x + 5 > 0

Проверка:

В процессе решения мы получили, что для всех найденных x:

3cos 2x − 2cos² x + 5 = 4

Поскольку 4 > 0, условие ОДЗ выполнено автоматически для всех решений.

✓ Все найденные значения x являются допустимыми
✓ Посторонних корней нет

📝 Ответ

x = π/4 + πk/2, k ∈ ℤ

Эквивалентная запись:

x = ±π/4 + πk, k ∈ ℤ

⚠️ Типичные ошибки

  • Забыли проверить ОДЗ: Для логарифмических уравнений обязательно проверять, что подлогарифмическое выражение положительно
  • Неправильное вычисление степени: (½)⁻² = 4, а не −4 или ¼. Отрицательный показатель меняет дробь на обратную
  • Потеряли знак ±: При извлечении корня из cos² x = ½ нужно рассматривать оба случая: cos x = √2/2 и cos x = −√2/2
  • Неправильная формула двойного угла: cos 2x = 2cos² x − 1, а не 2cos x − 1 или cos² x − 1
  • Ошибка в объединении серий: Четыре серии решений ±π/4 + 2πk и ±3π/4 + 2πk можно записать как одну: π/4 + πk/2
