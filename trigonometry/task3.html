Задача 3: Разность четвёртых степеней и формула приведения

Главная → Задания → Задача 3

Условие задачи

Решить тригонометрическое уравнение:

sin⁴(x/4) − cos⁴(x/4) = cos(x − π/2)

💡 Идея решения

  • Применить формулу разности четвёртых степеней: a⁴ − b⁴ = (a² − b²)(a² + b²)
  • Использовать основное тригонометрическое тождество: sin²α + cos²α = 1
  • Применить формулу приведения для правой части: cos(x − π/2) = sin x
  • Выразить sin x через половинный угол: sin x = 2·sin(x/2)·cos(x/2)
  • Вынести общий множитель и решить два простых уравнения
1Что дано

Нам дано уравнение, содержащее:

  • В левой части: разность четвёртых степеней синуса и косинуса от угла x/4
  • В правой части: косинус разности x − π/2

Обе части содержат тригонометрические функции, но от разных углов. Наша цель — привести всё к одному аргументу.

2Преобразование левой части

Применим формулу разности четвёртых степеней:

a⁴ − b⁴ = (a² − b²)(a² + b²)

В нашем случае a = sin(x/4), b = cos(x/4):

sin⁴(x/4) − cos⁴(x/4) = [sin²(x/4) − cos²(x/4)] · [sin²(x/4) + cos²(x/4)]

Вторая скобка равна 1 по основному тригонометрическому тождеству:

sin²(x/4) + cos²(x/4) = 1

Первая скобка — это косинус двойного угла с минусом:

sin²(x/4) − cos²(x/4) = −cos(2·x/4) = −cos(x/2)

Итог: левая часть упростилась до −cos(x/2)

3Преобразование правой части

Применим формулу приведения для косинуса:

cos(x − π/2) = cos(π/2 − x) = sin x

Это следует из того, что косинус — чётная функция, и из формулы приведения cos(π/2 − α) = sin α.

Итог: правая часть равна sin x

4Полученное уравнение

После преобразований исходное уравнение принимает вид:

−cos(x/2) = sin x

Теперь нужно привести обе части к одному аргументу. Выразим sin x через половинный угол:

sin x = 2·sin(x/2)·cos(x/2)

Подставляем:

−cos(x/2) = 2·sin(x/2)·cos(x/2)
5Перенос в одну сторону и вынесение общего множителя

Перенесём всё в левую часть:

2·sin(x/2)·cos(x/2) + cos(x/2) = 0

Выносим общий множитель cos(x/2):

cos(x/2) · [2·sin(x/2) + 1] = 0

Произведение равно нулю, когда хотя бы один из множителей равен нулю.

6Решение первого уравнения

Первый случай:

cos(x/2) = 0

Косинус равен нулю в точках π/2 + πk:

x/2 = π/2 + πk

Умножаем на 2:

x = π + 2πk,     k ∈ ℤ
7Решение второго уравнения

Второй случай:

2·sin(x/2) + 1 = 0

Отсюда:

sin(x/2) = −1/2

Синус равен −1/2 в двух сериях:

x/2 = −π/6 + 2πk    или    x/2 = 7π/6 + 2πk

Умножаем на 2:

x = −π/3 + 4πk    или    x = 7π/3 + 4πk,     k ∈ ℤ
8Проверка ответа

Проверка первой серии: x = π + 2πk

  • x/4 = π/4 + πk/2
  • При чётных k: sin⁴(π/4) − cos⁴(π/4) = (√2/2)⁴ − (√2/2)⁴ = 0
  • Правая часть: cos(π − π/2) = cos(π/2) = 0 ✓

Проверка второй серии: x = −π/3 + 4πk

  • x/4 = −π/12 + πk
  • Левая часть: −cos(−π/6) = −√3/2
  • Правая часть: sin(−π/3) = −√3/2 ✓

📝 Ответ

x = π + 2πk,

x = −π/3 + 4πk,
x = 7π/3 + 4πk,

k ∈ ℤ

⚠️ Типичные ошибки

Ошибка 1: Неправильное применение формулы разности четвёртых степеней.
Некоторые забывают вторую скобку (a² + b²) или неправильно раскрывают разность квадратов.
Ошибка 2: Потеря знака при применении формулы приведения.
cos(x − π/2) = sin x, а не −sin x. Важно помнить, что косинус — чётная функция.
Ошибка 3: Неправильный переход к половинному углу.
sin x = 2·sin(x/2)·cos(x/2), а не просто sin(x/2) или cos(x/2).
Ошибка 4: Пропуск одной из серий решений для sin(x/2) = −1/2.
Синус отрицателен в III и IV четвертях, поэтому нужны две серии ответов.
Ошибка 5: Ошибка при умножении на 2 при переходе от x/2 к x.
Не забудьте умножить весь период: 2πk превращается в 4πk.

📚 Тренировочные задания

Решите следующие уравнения по аналогии с разобранным примером:

Задание 1: sin⁴(x/4) − cos⁴(x/4) = 0

Ответ: x = π + 2πk, k ∈ ℤ
Идея: После упрощения получаем −cos(x/2)=0

Задание 2: sin⁴(x/4) − cos⁴(x/4) = −sin x

Ответ: x = π + 2πk, x = π/3 + 4πk, x = 5π/3 + 4πk, k ∈ ℤ
Идея: Знак правой части меняет серии для второй серии решений

Задание 3: sin⁴(x/4) − cos⁴(x/4) = cos(π/2 − x)

Ответ: x = π + 2πk, x = −π/3 + 4πk, x = 7π/3 + 4πk, k ∈ ℤ
Идея: cos(π/2 − x) = sin x, то есть то же уравнение

Задание 4: 2[sin⁴(x/4) − cos⁴(x/4)] = sin x

Ответ: x = π + 2πk, k ∈ ℤ
Идея: После упрощения вторая серия не имеет решений

Задание 5: sin⁴(x/4) − cos⁴(x/4) = ½·sin x

Ответ: x = π + 2πk, k ∈ ℤ
Идея: Коэффициент ½ меняет уравнение, остаётся одна серия
