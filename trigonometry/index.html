📐 Тригонометрические уравнения

Учебное пособие по алгебре для подготовки к ЕГЭ и углубленного изучения тригонометрии

📘 Введение

Это пособие посвящено устойчивым способам решения тригонометрических уравнений школьного уровня. Каждая разобранная задача проверена заново, а после каждого примера дана серия из пяти тренировочных упражнений с нарастающей сложностью.


Основной принцип работы: сначала точное преобразование, затем аккуратный переход к стандартному тригонометрическому уравнению и только после этого запись общего ответа. Во всех задачах решения ищутся в множестве действительных чисел.

🔑 Ключевые формулы

sin 2x = 2 sin x cos x
cos 2x = 1 − 2 sin² x = 2 cos² x − 1
sin(π/2 − x) = cos x
cos(x − π/2) = sin x
cos(π/2 + x) = −sin x
cos(3π/2 + x) = sin x
