Стереометрия: расстояние между скрещивающимися прямыми

Координатный метод — пошаговое решение с 3D-визуализацией

1 Теоретические основы

В пространстве две прямые могут быть параллельными, пересекающимися или скрещивающимися. Скрещивающиеся прямые не лежат в одной плоскости — они не пересекаются и не параллельны.

Пусть прямые заданы каноническими уравнениями:

L1 : x−x1 l1 = y−y1 m1 = z−z1 n1 L2 : x−x2 l2 = y−y2 m2 = z−z2 n2

где M1(x1,y1,z1) — точка на L1, а M2(x2,y2,z2) — точка на L2.

Направляющие векторы:

q→1 = ( l1 , m1 , n1 ) , q→2 = ( l2 , m2 , n2 )

Формула расстояния

Расстояние между скрещивающимися прямыми равно модулю проекции вектора M1M2→ на направление, перпендикулярное обеим прямым:

d = | ( M1M2→ , q→1 , q→2 ) | | q→1 × q→2 |

Числитель — смешанное произведение трёх векторов (объём параллелепипеда). Знаменатель — модуль векторного произведения направляющих векторов (площадь основания).

d = | x2−x1 y2−y1 z2−z1 l1 m1 n1 l2 m2 n2 | | q→1 × q→2 |
⚠ Важно: Если знаменатель равен нулю (векторное произведение нулевое), то прямые параллельны — используется другая формула. Если числитель равен нулю (при ненулевом знаменателе), прямые пересекаются и расстояние равно 0.

3 Алгоритм решения

  • Шаг 1. Записать координаты точек M1, M2 и направляющих векторов q→1, q→2.
  • Шаг 2. Найти вектор M1M2→=(x2−x1,y2−y1,z2−z1).
  • Шаг 3. Вычислить векторное произведение q→1×q→2 по формуле:
q→1 × q→2 = m1n2−n1m2 n1l2−l1n2 l1m2−m1l2
  • Шаг 4. Найти смешанное произведение через определитель 3×3.
  • Шаг 5. Подставить в формулу расстояния.
